Python'da Dekoratörler

Bu dersimizde Python'da dekoratörler nedir, nasıl tanımlanır ve kullanım örnekleri hakkında bilgi edineceksiniz.

Dekoratörler Nedir?

Dekoratörler, Python'da fonksiyonların davranışını değiştirmek için kullanılan güçlü araçlardır. Bir dekoratör, başka bir fonksiyonu alır, ona bazı ek özellikler ekler ve geriye bir fonksiyon döndürür.

Python'da dekoratörler, "@" sembolü ile kolayca tanımlanabilir. Bu, fonksiyonların davranışını değiştirmek için temiz ve etkili bir yol sağlar.

# Basit bir dekoratör örneği def selamla(func): def wrapper(): print("Fonksiyon çağrılmadan önce") func() print("Fonksiyon çağrıldıktan sonra") return wrapper @selamla def merhaba(): print("Merhaba Dünya!") # Fonksiyonu çağıralım merhaba()

Biliyor muydunuz?

Dekoratörler, Python'da fonksiyonel programlamanın önemli bir parçasıdır ve "Metaprogramlama" olarak adlandırılan bir programlama tekniğini mümkün kılar.

Dekoratör Temelleri

Dekoratörleri anlamak için, Python'da fonksiyonların birinci sınıf nesneler olduğunu bilmek önemlidir:

  • Fonksiyonlar değişkenlere atanabilir.
  • Fonksiyonlar başka fonksiyonlara parametre olarak geçilebilir.
  • Fonksiyonlar başka fonksiyonlardan döndürülebilir.
  • Fonksiyonlar veri yapıları içinde saklanabilir.
# Fonksiyonların birinci sınıf nesneler olduğuna dair örnekler def topla(a, b): return a + b # Fonksiyonu bir değişkene atama hesapla = topla print(hesapla(5, 3)) # 8 # Fonksiyonu başka bir fonksiyona parametre olarak geçirme def islem_yap(fonksiyon, x, y): return fonksiyon(x, y) print(islem_yap(topla, 10, 20)) # 30 # Fonksiyondan fonksiyon döndürme def islem_sec(operation): if operation == "topla": return topla else: return lambda x, y: x * y secilen_islem = islem_sec("topla") print(secilen_islem(7, 8)) # 15

Dekoratörlerin Çalışma Mantığı

Bir dekoratör, temel olarak şu adımları gerçekleştirir:

1. Bir Fonksiyonu Parametre Olarak Alır

Dekoratör, dekore edeceği fonksiyonu parametre olarak alır.

def dekorator(func): # İç fonksiyon (wrapper) def wrapper(): # Fonksiyon çağrılmadan önce yapılacak işlemler print("Fonksiyon çağrılmadan önce") # Orijinal fonksiyonu çağır func() # Fonksiyon çağrıldıktan sonra yapılacak işlemler print("Fonksiyon çağrıldıktan sonra") # Wrapper fonksiyonunu döndür return wrapper

2. İçinde Başka Bir Fonksiyon Tanımlar (Wrapper)

Bu iç fonksiyon (genellikle wrapper olarak adlandırılır), orijinal fonksiyonu çağıran ve ek işlevsellik ekleyen bir fonksiyondur.

3. Wrapper Fonksiyonunu Döndürür

Dekoratör, bu wrapper fonksiyonunu döndürür ve orijinal fonksiyon bu wrapper fonksiyonu ile değiştirilir.

# Adım adım dekoratör kullanımı def dekorator(func): def wrapper(): print("Ekstra işlevsellik başladı") func() print("Ekstra işlevsellik bitti") return wrapper # Manuel olarak dekoratör kullanımı def selam_ver(): print("Merhaba!") # Fonksiyonu dekore etme dekore_edilmis = dekorator(selam_ver) dekore_edilmis() # Bu çağrı wrapper'ı çalıştırır # @ sembolü ile aynı işlem @dekorator def selam_ver2(): print("Merhaba!") selam_ver2() # Bu da wrapper'ı çalıştırır

Parametreli Fonksiyonlar için Dekoratörler

Gerçek uygulamalarda, fonksiyonlarınız genellikle parametreler alır. Dekoratörlerinizin bu parametreleri ele alabilmesi gerekir:

# Parametreli fonksiyonlar için dekoratör def parametreli_dekorator(func): def wrapper(*args, **kwargs): print(f"Fonksiyon çağrılıyor: {func.__name__}") print(f"Argümanlar: {args}, Anahtar kelimeli argümanlar: {kwargs}") return func(*args, **kwargs) return wrapper @parametreli_dekorator def topla(a, b): return a + b @parametreli_dekorator def kisi_bilgisi(ad, soyad, yas=None): if yas: return f"{ad} {soyad}, {yas} yaşında" else: return f"{ad} {soyad}" print(topla(10, 20)) # 30 print(kisi_bilgisi("Ali", "Yılmaz", yas=30)) # Ali Yılmaz, 30 yaşında

Dönüş Değerini Koruma

Yukarıdaki örnekte, wrapper fonksiyonunun, orijinal fonksiyonun dönüş değerini döndürdüğüne dikkat edin. Bu, dekoratörün fonksiyonun davranışını değiştirirken, dönüş değerini korumasını sağlar.

# Dönüş değeriyle ilgili bir dekoratör örneği def sonucu_iki_katina_cikar(func): def wrapper(*args, **kwargs): sonuc = func(*args, **kwargs) return sonuc * 2 return wrapper @sonucu_iki_katina_cikar def sayi_al(): return 5 @sonucu_iki_katina_cikar def toplam(sayilar): return sum(sayilar) print(sayi_al()) # 10 (5*2) print(toplam([1, 2, 3, 4])) # 20 ((1+2+3+4)*2)

Popüler Dekoratör Örnekleri

1. Zamanlama Dekoratörü

  • İşlev: Bir fonksiyonun çalışma süresini ölçmek için kullanılır.
import time def zaman_olc(func): def wrapper(*args, **kwargs): baslangic = time.time() sonuc = func(*args, **kwargs) bitis = time.time() print(f"{func.__name__} fonksiyonu {bitis - baslangic:.4f} saniyede çalıştı.") return sonuc return wrapper @zaman_olc def uzun_islem(): """Uzun süren bir işlemi simüle eder""" time.sleep(2) return "İşlem tamamlandı" uzun_islem() # uzun_islem fonksiyonu 2.0004 saniyede çalıştı.

2. Loglama Dekoratörü

  • İşlev: Fonksiyon çağrılarını, parametrelerini ve sonuçlarını loglar.
def logla(func): def wrapper(*args, **kwargs): print(f"ÇAĞRI: {func.__name__} fonksiyonu çağrıldı.") print(f"PARAMETRELER: args={args}, kwargs={kwargs}") sonuc = func(*args, **kwargs) print(f"SONUÇ: {sonuc}") return sonuc return wrapper @logla def kare_al(sayi): return sayi ** 2 kare_al(5) # ÇAĞRI: kare_al fonksiyonu çağrıldı. # PARAMETRELER: args=(5,), kwargs={} # SONUÇ: 25

3. Önbellek (Cache) Dekoratörü

  • İşlev: Fonksiyon sonuçlarını önbellekler, tekrar hesaplamayı önler.
def onbellek(func): cache = {} # Sonuçları saklayacak sözlük def wrapper(*args): # Argümanlar değişebilir veri türü içerebileceği için tuple'a dönüştürüyoruz cache_key = args # Eğer bu argümanlarla daha önce hesaplanmışsa, önbellekten döndür if cache_key in cache: print(f"Önbellekten alınıyor: {args}") return cache[cache_key] # Hesapla ve önbelleğe kaydet sonuc = func(*args) cache[cache_key] = sonuc print(f"Hesaplanıyor ve önbelleğe alınıyor: {args}") return sonuc return wrapper @onbellek def fibonacci(n): """Fibonacci serisinin n. elemanını döndürür""" if n <= 1: return n return fibonacci(n-1) + fibonacci(n-2) # İlk çağrı hesaplanacak print(fibonacci(10)) # 55 # İkinci çağrı önbellekten alınacak print(fibonacci(10)) # 55

4. Yetkilendirme Dekoratörü

  • İşlev: Fonksiyona erişim yetkisini kontrol eder.
def yetki_kontrol(role): def decorator(func): def wrapper(*args, **kwargs): # Gerçek uygulamada burada kullanıcı kimliği ve rolü kontrol edilir user_role = "admin" # Örnek olarak varsayılan bir rol if user_role != role: raise PermissionError(f"Bu işlemi yapmak için '{role}' rolüne sahip olmanız gerekiyor.") return func(*args, **kwargs) return wrapper return decorator @yetki_kontrol("admin") def gizli_verileri_getir(): return "Gizli veriler: XYZ123" # Bu çalışacak çünkü user_role "admin" olarak ayarlandı print(gizli_verileri_getir()) # Eğer user_role değiştirilirse (örneğin "user" olarak), PermissionError hatası alınacaktır

Fonksiyon Metadata'sını Koruma

Dekoratörler kullandığınızda, orijinal fonksiyonun adı, dokümantasyonu ve diğer meta verilerini kaybedebilirsiniz. Python'un functools modülündeki @wraps dekoratörü bu sorunu çözer.

from functools import wraps def ornek_dekorator(func): @wraps(func) # Orijinal fonksiyonun meta verilerini korur def wrapper(*args, **kwargs): """Bu wrapper fonksiyonudur""" return func(*args, **kwargs) return wrapper @ornek_dekorator def ornek_fonksiyon(): """Bu örnek bir fonksiyondur""" pass # @wraps kullanılmazsa, aşağıdaki çıktı "wrapper" olacaktı print(ornek_fonksiyon.__name__) # "ornek_fonksiyon" # @wraps kullanılmazsa, aşağıdaki çıktı wrapper'ın dokümantasyonu olacaktı print(ornek_fonksiyon.__doc__) # "Bu örnek bir fonksiyondur"

Yapılabilecek Projeler

Python'da dekoratörler konusunda pratik yapmak için aşağıdaki projeleri deneyebilirsiniz:

Performans Analiz Aracı

Farklı fonksiyonların çalışma sürelerini ölçen ve karşılaştıran bir performans analiz dekoratörü yazın.

Başlangıç

Giriş Doğrulama Sistemi

Web uygulamanızda kullanıcı girişi gerektiren fonksiyonlar için yetkilendirme dekoratörleri geliştirin.

Başlangıç

Tekrar Denemeli Fonksiyonlar

Hata durumunda belirli sayıda tekrar deneme yapan bir dekoratör yazarak ağ istekleri gibi başarısız olabilecek işlemleri güvenli hale getirin.

Başlangıç

Veritabanı İşlem Dekoratörü

Veritabanı bağlantılarını açıp kapatan ve işlemleri (transaction) yöneten bir dekoratör yazarak veritabanı uygulamalarını daha güvenli hale getirin.

Sonraki Adımlar

Python decorator'lar konusunu tamamladıktan sonra öğrenmenize devam edebileceğiniz konular:

  • Generators: İteratörler ve generator'ları kullanarak bellek verimli kodlar yazın.
  • Context Managers: with ifadesi ve __enter__/__exit__ metodlarını öğrenin.
  • Meta Programming: Çalışma zamanında programların davranışını değiştirmeyi öğrenin.
  • Functional Programming: Lambda, map, filter ve reduce gibi fonksiyonel programlama tekniklerini öğrenin.
  • Design Patterns: Python'da yaygın olarak kullanılan tasarım desenlerini öğrenin.

Başarı İpucu

Python'da decorator'ları öğrenirken, önce basit decorator'lar yazarak başlayın, sonra parametreli decorator'lara geçin. Gerçek dünya uygulamaları üzerinde çalışın: zamanlama, loglama, önbellekleme (caching) ve yetkilendirme gibi senaryolar için decorator'lar yazmayı deneyin!
