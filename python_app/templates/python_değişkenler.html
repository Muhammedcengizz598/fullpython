Python'da Değişkenler ve Veri Tipleri

Bu dersimizde Python'da değişkenler nedir, nasıl tanımlanır ve farklı veri tipleri hakkında bilgi edineceksiniz.

Değişkenler Nedir?

Değişkenler, programlama dillerinde veri saklamak için kullanılan sembolik isimlerdir. Bir değişken, programınızda daha sonra kullanmak üzere bir değeri hafızada tutan bir konteynır gibi düşünülebilir.

Python'da değişken tanımlamak için özel bir anahtar kelime kullanmanıza gerek yoktur. Sadece değişken adını yazıp ardından eşittir (=) işareti ile değer atayabilirsiniz.

# Değişken tanımlama isim = "Ahmet" yas = 25 boy = 1.78 ogrenci_mi = True # Değişkenleri kullanma print("Merhaba, benim adım", isim) print("Yaşım:", yas) print("Boyum:", boy, "metre") print("Öğrenci miyim:", ogrenci_mi)

Biliyor muydunuz?

Python'da değişken adları büyük/küçük harfe duyarlıdır. Yani "yas", "YAS" ve "Yas" farklı değişkenlerdir.

Değişken İsimlendirme Kuralları

Python'da değişken isimlendirme için bazı kurallar vardır:

  • Değişken adı bir harf veya alt çizgi (_) ile başlamalıdır.
  • Değişken adı sayı ile başlayamaz.
  • Değişken adı sadece alfanümerik karakterler ve alt çizgi içerebilir (A-z, 0-9 ve _).
  • Değişken adları boşluk içeremez.
  • Python'un anahtar kelimelerini (if, else, for, while, def, class vb.) değişken adı olarak kullanamazsınız.
# Geçerli değişken adları isim = "Ali" _isim = "Veli" isim1 = "Ayşe" isim_soyisim = "Fatma Demir" # Geçersiz değişken adları # 1isim = "Hata!" # Sayı ile başlayamaz # isim-soyisim = "Hata!" # Tire işareti kullanılamaz # if = "Hata!" # Anahtar kelime kullanılamaz

Python'da Veri Tipleri

Python'da temel veri tipleri şunlardır:

1. Sayısal Veri Tipleri

  • int (integer): Tam sayılar (10, -5, 0)
  • float: Ondalık sayılar (3.14, -0.5, 2.0)
  • complex: Karmaşık sayılar (3+5j)
# Integer (Tam Sayı) x = 10 print(x, "tipinde:", type(x)) # Float (Ondalık Sayı) y = 3.14 print(y, "tipinde:", type(y)) # Complex (Karmaşık Sayı) z = 3+5j print(z, "tipinde:", type(z))

2. Metin Veri Tipi

  • str (string): Metinsel ifadeler ("Merhaba", 'Python', "123")
# String (Metin) ad = "Python" print(ad, "tipinde:", type(ad)) # Farklı tırnak işaretleri kullanabilirsiniz mesaj1 = 'Tek tırnak içinde' mesaj2 = "Çift tırnak içinde" mesaj3 = '''Üç tırnak içinde çok satırlı metin yazabilirsiniz''' print(mesaj1) print(mesaj2) print(mesaj3)

3. Boolean Veri Tipi

  • bool: True veya False değerlerini alır
# Boolean dogru = True yanlis = False print(dogru, "tipinde:", type(dogru)) print(yanlis, "tipinde:", type(yanlis)) # Karşılaştırmalar boolean sonuç üretir print("5 > 3:", 5 > 3) # True print("5 == 3:", 5 == 3) # False

4. Sıralı (Sequence) Veri Tipleri

  • list: Sıralı ve değiştirilebilir koleksiyon [1, "elma", 3.14]
  • tuple: Sıralı ve değiştirilemez koleksiyon (1, "elma", 3.14)
  • range: Sayı dizisini temsil eder, genellikle döngülerde kullanılır
# List (Liste) meyveler = ["elma", "armut", "muz"] print(meyveler, "tipinde:", type(meyveler)) # Listenin elemanlarına erişim print("İlk meyve:", meyveler[0]) print("Son meyve:", meyveler[-1]) # Listeyi değiştirme meyveler[1] = "çilek" print("Değiştirilmiş liste:", meyveler) # Tuple (Demet) koordinatlar = (10, 20) print(koordinatlar, "tipinde:", type(koordinatlar)) # Tuple'ın elemanlarına erişim print("x koordinatı:", koordinatlar[0]) print("y koordinatı:", koordinatlar[1]) # Tuple değiştirilemez (immutable) # koordinatlar[0] = 15 # Bu satır hata verecektir # Range sayilar = range(5) # 0, 1, 2, 3, 4 print(sayilar, "tipinde:", type(sayilar)) print("Range'deki sayılar:", list(sayilar))

5. Eşleme (Mapping) Veri Tipi

  • dict (dictionary): Anahtar-değer çiftlerinden oluşan koleksiyon {"ad": "Ali", "yas": 25}
# Dictionary (Sözlük) kisi = { "ad": "Ali", "yas": 25, "meslek": "Mühendis" } print(kisi, "tipinde:", type(kisi)) # Sözlüğün değerlerine erişim print("Ad:", kisi["ad"]) print("Yaş:", kisi["yas"]) # Sözlüğü değiştirme kisi["yas"] = 26 print("Güncellenmiş yaş:", kisi["yas"]) # Yeni anahtar-değer çifti ekleme kisi["sehir"] = "İstanbul" print("Güncellenmiş sözlük:", kisi)

6. Set Veri Tipleri

  • set: Sırasız ve benzersiz elemanlardan oluşan koleksiyon {1, 2, 3}
  • frozenset: Değiştirilemez set
# Set renkler = {"kırmızı", "mavi", "yeşil"} print(renkler, "tipinde:", type(renkler)) # Set'e eleman ekleme renkler.add("sarı") print("Güncellenmiş set:", renkler) # Set'te aynı eleman tekrar eklenemez renkler.add("mavi") # Bu işlem seti değiştirmez print("Tekrar eklemeden sonra:", renkler) # Frozenset sabit_renkler = frozenset(["siyah", "beyaz"]) print(sabit_renkler, "tipinde:", type(sabit_renkler)) # Frozenset değiştirilemez # sabit_renkler.add("gri") # Bu satır hata verecektir

7. None Veri Tipi

  • None: Değer yokluğunu temsil eder
# None deger = None print(deger, "tipinde:", type(deger)) # None kontrolü if deger is None: print("Değer tanımlanmamış") else: print("Değer tanımlanmış")

Veri Tipi Dönüşümleri

Python'da bir veri tipinden diğerine dönüşüm yapabilirsiniz:

# Int'den diğer tiplere dönüşüm sayi = 10 print("Int:", sayi) print("Float'a dönüşüm:", float(sayi)) print("String'e dönüşüm:", str(sayi)) print("Boolean'a dönüşüm:", bool(sayi)) # String'den diğer tiplere dönüşüm metin = "123" print("String:", metin) print("Int'e dönüşüm:", int(metin)) print("Float'a dönüşüm:", float(metin)) # Boolean'dan diğer tiplere dönüşüm b = True print("Boolean:", b) print("Int'e dönüşüm:", int(b)) # True -> 1 print("String'e dönüşüm:", str(b)) # True -> "True" # Dönüşüm hataları # int("abc") # ValueError: invalid literal for int() with base 10: 'abc'

Tip Kontrolü

Python'da bir değişkenin tipini kontrol etmek için type() fonksiyonu kullanılır. Belirli bir tipte olup olmadığını kontrol etmek için ise isinstance() fonksiyonu kullanılabilir.

x = 10 print("x'in tipi:", type(x)) print("x bir integer mi?", isinstance(x, int)) print("x bir string mi?", isinstance(x, str))

Başlangıç

Birim Dönüştürücü

Başlangıç

Adres Defteri

Listeler ve sözlükler kullanarak telefon numaralarını ve e-posta adreslerini saklayabileceğiniz bir adres defteri uygulaması geliştirin.

1. Python'da değişken isimlendirme kurallarına göre, aşağıdakilerden hangisi geçerli bir değişken adı DEĞİLDİR?

A
_isim
B
isim_soyisim
C
1isim
D
isim1
Doğru! Python'da değişken adları sayı ile başlayamaz.
Yanlış. Python'da değişken adları sayı ile başlayamaz, bu nedenle "1isim" geçerli bir değişken adı değildir.

2. Aşağıdaki Python kodunun çıktısı ne olur?

x = 5
y = "10"
print(x + int(y))
A
"510"
B
15
C
"15"
D
TypeError
Doğru! String "10" değeri int() fonksiyonu ile tamsayı 10'a dönüştürülür ve 5 + 10 = 15 sonucunu verir.
Yanlış. String "10" değeri int() fonksiyonu ile tamsayı 10'a dönüştürülür ve 5 + 10 = 15 sonucunu verir.

3. Aşağıdaki veri tiplerinden hangisi değiştirilemez (immutable) değildir?

A
int
B
str
C
tuple
D
list
Doğru! Listeler (list) değiştirilebilir (mutable) veri tipidir. int, str ve tuple değiştirilemez (immutable) veri tipleridir.
Yanlış. Listeler (list) değiştirilebilir (mutable) veri tipidir. int, str ve tuple değiştirilemez (immutable) veri tipleridir.

4. Aşağıdaki koddaki hata nedir?

tuple_ornek = (1, 2, 3)
tuple_ornek[0] = 5
A

B
TypeError
C
ValueError
D
SyntaxError
Doğru! Tuple'lar değiştirilemez (immutable) olduğu için, bir tuple elemanını değiştirmeye çalıştığınızda TypeError hatası alırsınız.
Yanlış. Tuple'lar değiştirilemez (immutable) olduğu için, bir tuple elemanını değiştirmeye çalıştığınızda TypeError hatası alırsınız.

Doğru! bool(0), bool("") ve bool([]) ifadelerinin hepsi False değerini döndürür. False or False or False işleminin sonucu False'dir.
Yanlış. bool(0), bool("") ve bool([]) ifadelerinin hepsi False değerini döndürür. False or False or False işleminin sonucu False'dir.

Yanlış. Sözlük tanımlamada anahtar ve değer çiftleri, iki nokta üst üste (:) ile ayrılır ve süslü parantezler içinde yazılır.

7. Python'da değişken tanımlarken doğru olan hangisidir?

A

Yanlış. Python'da değişken adları alt çizgi (_) ile başlayabilir, harflerle başlayabilir ama sayı ile başlayamaz ve boşluk veya özel karakterler içeremez.
