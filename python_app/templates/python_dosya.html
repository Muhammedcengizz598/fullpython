Python'da Dosya İşlemleri

Bu dersimizde Python'da dosyalar nasıl açılır, okunur, yazılır ve dosya işlemleri hakkında temel bilgiler edineceksiniz.

Dosya İşlemleri Nedir?

Dosya işlemleri, programların bilgisayardaki dosyalara erişmesini, onları okumasını ve onlara yazmasını sağlayan işlemlerdir. Python, dosya işlemleri için oldukça kullanışlı ve kolay bir arayüz sunar.

Python'da dosya işlemleri built-in (yerleşik) fonksiyonlar ve metotlar ile gerçekleştirilir. Dosyalarla çalışmak, veri depolama, veri okuma, loglar oluşturma gibi birçok temel programlama görevinde önemlidir.

# Basit bir dosya okuma örneği dosya = open("ornek.txt", "r") icerik = dosya.read() print(icerik) dosya.close() # Daha güvenli bir yaklaşım: with ifadesi with open("ornek.txt", "r") as dosya: icerik = dosya.read() print(icerik) # Dosya otomatik olarak kapatılır

Biliyor muydunuz?

Python'da "with" ifadesi kullanarak dosya işlemlerini yapmak, dosyanın otomatik olarak kapatılmasını sağlar ve olası hatalarda bile dosyanın düzgün kapanmasını garantiler.

Dosya Açma Modları

Python'da dosyaları açmak için farklı modlar vardır:

  • "r": Okuma modu (default). Dosya yoksa hata verir.
  • "w": Yazma modu. Dosya yoksa oluşturur, varsa içeriğini siler.
  • "a": Ekleme modu. Dosya yoksa oluşturur, varsa sonuna ekleme yapar.
  • "x": Özel oluşturma modu. Dosya zaten varsa hata verir.
  • "b": İkili (binary) mod. Metin yerine ikili veri için kullanılır (örn. "rb", "wb").
  • "t": Metin modu (default). Metin verileriyle çalışmak için kullanılır.
  • "+": Okuma ve yazma için güncelleme modu (örn. "r+", "w+").
# Farklı modlarda dosya açma örnekleri # Okuma modu with open("ornek.txt", "r") as dosya: icerik = dosya.read() # Yazma modu with open("yeni.txt", "w") as dosya: dosya.write("Merhaba Dünya!") # Ekleme modu with open("log.txt", "a") as dosya: dosya.write("\nYeni bir log satırı eklendi.") # İkili mod (resim dosyası gibi) with open("resim.jpg", "rb") as dosya: icerik = dosya.read() # Hem okuma hem yazma modu with open("veri.txt", "r+") as dosya: veri = dosya.read() dosya.write("Yeni veri")

Dosya Okuma Yöntemleri

Python'da dosya içeriğini okumak için birkaç farklı yöntem vardır:

1. read() Metodu

  • read(): Tüm dosya içeriğini tek bir string olarak okur.
  • read(n): Belirtilen sayıda karakter okur.
# Tüm dosyayı okuma with open("ornek.txt", "r") as dosya: icerik = dosya.read() print(icerik) # Belirli sayıda karakter okuma with open("ornek.txt", "r") as dosya: ilk_10_karakter = dosya.read(10) print("İlk 10 karakter:", ilk_10_karakter) # Dosya işaretçisi kaldığı yerden devam eder sonraki_5_karakter = dosya.read(5) print("Sonraki 5 karakter:", sonraki_5_karakter)

2. readline() Metodu

  • readline(): Dosyadan bir satır okur.
# Satır satır okuma with open("ornek.txt", "r") as dosya: ilk_satir = dosya.readline() print("İlk satır:", ilk_satir) ikinci_satir = dosya.readline() print("İkinci satır:", ikinci_satir) # readline() ile tüm dosyayı satır satır okuma with open("ornek.txt", "r") as dosya: satir = dosya.readline() while satir: print(satir, end='') # Her satırın sonunda zaten \n var satir = dosya.readline()

3. readlines() Metodu

  • readlines(): Tüm satırları bir liste olarak okur.
# Tüm satırları liste olarak okuma with open("ornek.txt", "r") as dosya: satirlar = dosya.readlines() print("Toplam satır sayısı:", len(satirlar)) for i, satir in enumerate(satirlar): print(f"{i+1}. satır: {satir}", end='')

4. Dosya Objesini İterasyon ile Okuma

  • for döngüsü: Dosya objesi üzerinde doğrudan iterasyon yapabilirsiniz.
# Dosya üzerinde iterasyon - en verimli yöntem with open("ornek.txt", "r") as dosya: for satir in dosya: print(satir, end='')

Dosya Okuma İpucu

Büyük dosyaları okurken for satir in dosya yaklaşımı bellekte tüm dosyanın tutulmasını gerektirmediği için en verimli yöntemdir.

# Büyük dosya işleme örneği with open("buyuk_veri.txt", "r") as dosya: # Her satırı tek tek işle toplam_kelime = 0 for satir in dosya: kelimeler = satir.split() toplam_kelime += len(kelimeler) print(f"Dosyadaki toplam kelime sayısı: {toplam_kelime}")

Dosyaya Yazma

Python'da dosyaya yazmak için birkaç farklı metot vardır:

1. write() Metodu

  • write(): Verilen string'i dosyaya yazar.
# Dosyaya yazma with open("cikti.txt", "w") as dosya: dosya.write("Merhaba, Python!") dosya.write("Bu ikinci cümle.") # Otomatik satır sonu eklenmez # Satır sonu karakterleri ekleyerek yazma with open("cikti2.txt", "w") as dosya: dosya.write("Birinci satır\n") dosya.write("İkinci satır\n") dosya.write("Üçüncü satır\n")

2. writelines() Metodu

  • writelines(): String listesini dosyaya yazar.
# Birden çok satırı liste olarak yazma satirlar = ["Birinci satır\n", "İkinci satır\n", "Üçüncü satır\n"] with open("cikti3.txt", "w") as dosya: dosya.writelines(satirlar) # Liste hazırlayıp yazma isimler = ["Ali", "Veli", "Ayşe", "Fatma"] with open("isimler.txt", "w") as dosya: for isim in isimler: dosya.write(isim + "\n")

Dosya Konumu ve Arama

Dosya işlemlerinde, dosya içindeki konumu kontrol etmek ve değiştirmek için çeşitli metotlar vardır:

# Dosya konumunu öğrenme ve değiştirme with open("ornek.txt", "r") as dosya: # tell() mevcut konumu gösterir print("Başlangıç konumu:", dosya.tell()) # İlk 10 karakteri oku icerik = dosya.read(10) print("Okunan içerik:", icerik) print("Yeni konum:", dosya.tell()) # seek() ile konumu değiştir dosya.seek(0) # Dosyanın başına dön print("Başa döndükten sonra konum:", dosya.tell()) # Dosyadan tekrar oku icerik = dosya.read(5) print("Yeniden okunan içerik:", icerik) # Belirli bir konuma git dosya.seek(20) print("20. konumdan sonraki içerik:", dosya.read(10))

Dosya ve Dizin İşlemleri

Python'da dosya ve dizin işlemleri için os ve shutil modülleri kullanılır:

import os import shutil # Dosya varlığını kontrol etme if os.path.exists("ornek.txt"): print("Dosya mevcut") else: print("Dosya bulunamadı") # Dosya mı klasör mü kontrol etme if os.path.isfile("ornek.txt"): print("Bu bir dosya") elif os.path.isdir("ornek_klasor"): print("Bu bir klasör") # Dosya adı ve dizin bilgisi alma dosya_yolu = "/home/kullanici/belgeler/ornek.txt" print("Dosya adı:", os.path.basename(dosya_yolu)) print("Dizin adı:", os.path.dirname(dosya_yolu)) # Klasör oluşturma os.mkdir("yeni_klasor") # İç içe klasör oluşturma os.makedirs("ana_klasor/alt_klasor/daha_alt") # Mevcut dizini öğrenme print("Mevcut dizin:", os.getcwd()) # Dizini değiştirme os.chdir("yeni_klasor") # Dizin içeriğini listeleme print("Dizin içeriği:", os.listdir()) # Dosya silme os.remove("silinecek.txt") # Klasör silme os.rmdir("bos_klasor") # Sadece boş klasörler silinebilir # Dolu klasör silme shutil.rmtree("dolu_klasor") # İçeriğiyle birlikte siler # Dosya kopyalama shutil.copy("kaynak.txt", "hedef.txt") # Dosya taşıma/yeniden adlandırma os.rename("eski_ad.txt", "yeni_ad.txt") shutil.move("kaynak.txt", "yeni_konum/hedef.txt")

CSV Dosya İşlemleri

Python'da CSV (Comma-Separated Values) dosyalarıyla çalışmak için csv modülü kullanılır:

import csv # CSV dosyası yazma with open('veriler.csv', 'w', newline='') as dosya: yazici = csv.writer(dosya) yazici.writerow(['Ad', 'Soyad', 'Yaş']) yazici.writerow(['Ali', 'Yılmaz', 25]) yazici.writerow(['Ayşe', 'Demir', 30]) yazici.writerow(['Mehmet', 'Kaya', 22]) # CSV dosyası okuma with open('veriler.csv', 'r') as dosya: okuyucu = csv.reader(dosya) for satir in okuyucu: print(satir) # DictReader ve DictWriter kullanımı with open('kullanicilar.csv', 'w', newline='') as dosya: alanlar = ['ad', 'email', 'sehir'] yazici = csv.DictWriter(dosya, fieldnames=alanlar) yazici.writeheader() yazici.writerow({'ad': 'Ali', 'email': 'ali@ornek.com', 'sehir': 'İstanbul'}) yazici.writerow({'ad': 'Veli', 'email': 'veli@ornek.com', 'sehir': 'Ankara'}) # DictReader ile okuma with open('kullanicilar.csv', 'r') as dosya: okuyucu = csv.DictReader(dosya) for satir in okuyucu: print(f"{satir['ad']} - {satir['email']} - {satir['sehir']}")

JSON Dosya İşlemleri

Python'da JSON (JavaScript Object Notation) dosyalarıyla çalışmak için json modülü kullanılır:

import json # Python sözlüğü kullanici = { "id": 1, "ad": "Ali Yılmaz", "email": "ali@ornek.com", "aktif": True, "roller": ["admin", "editor"], "adres": { "sokak": "Atatürk Cad.", "sehir": "İstanbul", "ulke": "Türkiye" } } # JSON dosyasına yazma with open("kullanici.json", "w") as dosya: json.dump(kullanici, dosya, indent=4) # JSON dosyasını okuma with open("kullanici.json", "r") as dosya: okunan_kullanici = json.load(dosya) print(okunan_kullanici["ad"]) print(okunan_kullanici["adres"]["sehir"]) # String to JSON json_string = '{"ad": "Veli", "yas": 30}' veri = json.loads(json_string) print(veri["ad"]) # JSON to string yeni_json_string = json.dumps(kullanici, indent=4) print(yeni_json_string)
