Python'da Girdi ve Çıktı İşlemleri

Bu dersimizde Python'da girdi alma, ekrana çıktı verme ve dosya işlemleri hakkında bilgi edineceksiniz.

Kullanıcıdan Girdi Alma

Python'da kullanıcıdan girdi almak için input() fonksiyonu kullanılır. Bu fonksiyon, kullanıcıya bir mesaj gösterir ve kullanıcının klavyeden bir değer girmesini bekler.

input() fonksiyonu her zaman string (metin) türünde değer döndürür. Sayısal bir değer almak istiyorsanız, dönüştürme işlemi yapmanız gerekir.

# Basit girdi alma isim = input("Adınız: ") print("Merhaba,", isim) # Sayısal girdi alma yas_str = input("Yaşınız: ") yas = int(yas_str) # String'i integer'a dönüştürme # Tek satırda dönüştürme yapma boy = float(input("Boyunuz (metre cinsinden): ")) # Birden fazla girdi alma sayi1 = int(input("Birinci sayı: ")) sayi2 = int(input("İkinci sayı: ")) toplam = sayi1 + sayi2 print("Toplam:", toplam)

Biliyor muydunuz?

Python 2 ile Python 3 arasında girdi alma fonksiyonları farklıdır. Python 2'de raw_input() ve input() olmak üzere iki farklı fonksiyon vardı. Python 3'te ise sadece input() fonksiyonu bulunur ve Python 2'deki raw_input() fonksiyonu gibi davranır.

Ekrana Çıktı Verme

Python'da ekrana çıktı vermek için en yaygın kullanılan fonksiyon print() fonksiyonudur. Bu fonksiyon, verilen değerleri ekrana yazdırır ve varsayılan olarak sonuna yeni satır ekler.

# Temel print kullanımı print("Merhaba, Dünya!") # Birden fazla değeri yazdırma ad = "Ali" yas = 25 print("Ad:", ad, "Yaş:", yas) # Değerleri birleştirerek yazdırma print("Ad: " + ad + ", Yaş: " + str(yas)) # f-string kullanımı (Python 3.6+) print(f"Ad: {ad}, Yaş: {yas}") # format() metodu ile biçimlendirme print("Ad: {}, Yaş: {}".format(ad, yas)) print("Ad: {0}, Yaş: {1}".format(ad, yas)) print("Ad: {ad}, Yaş: {yas}".format(ad=ad, yas=yas)) # Sayısal değerleri biçimlendirme pi = 3.14159265359 print(f"Pi sayısı: {pi:.2f}") # Ondalık kısmı 2 basamağa sınırlama print("Pi sayısı: {:.4f}".format(pi)) # Ondalık kısmı 4 basamağa sınırlama # Özel karakterler print("Birinci satır\nİkinci satır") # \n yeni satır ekler print("Ad:\tAli") # \t tab karakteri ekler print("C:\\Users\\Ali") # Ters eğik çizgi için çift ters eğik çizgi # sep ve end parametreleri print("Elma", "Armut", "Muz", sep=", ") # Elemanlar arasına virgül ve boşluk koyar print("İşlem devam ediyor", end="... ") # Sonuna üç nokta koyar ve yeni satıra geçmez print("tamamlandı!")

print() Fonksiyonunun Parametreleri

  • sep: Yazdırılan değerler arasında kullanılacak ayırıcı (varsayılan: boşluk)
  • end: Yazdırma işlemi sonunda eklenecek karakter (varsayılan: '\n', yeni satır)
  • file: Çıktının yönlendirileceği dosya nesnesi (varsayılan: sys.stdout)
  • flush: Çıktının hemen yazılıp yazılmayacağını belirler (varsayılan: False)
# sep parametresi print("Elma", "Armut", "Muz") # Elma Armut Muz print("Elma", "Armut", "Muz", sep="-") # Elma-Armut-Muz # end parametresi print("Merhaba", end=" ") print("Dünya") # Merhaba Dünya # file parametresi import sys print("Hata mesajı", file=sys.stderr) # Standart hata akışına yazdırma # flush parametresi import time for i in range(5): print(i, end=" ", flush=True) # Hemen ekrana yazdırır time.sleep(1)

String Biçimlendirme (String Formatting)

Python'da metinleri biçimlendirmek için çeşitli yöntemler bulunmaktadır:

1. f-string (Python 3.6+)

f-string, Python 3.6 ve sonraki sürümlerinde kullanılabilen, kolay ve okunabilir string biçimlendirme yöntemidir.

ad = "Ahmet" yas = 30 boy = 1.75 # Temel f-string kullanımı print(f"Merhaba, benim adım {ad}, {yas} yaşındayım ve boyum {boy} metre.") # Hesaplama yapma print(f"5 yıl sonra {yas + 5} yaşında olacağım.") # Sayısal değerleri biçimlendirme print(f"Pi sayısı: {3.14159265359:.2f}") # 3.14 print(f"Yüzde: {0.75:.0%}") # 75% # Sağa/sola yaslama ve genişlik belirtme print(f"|{ad:<10}|") # Sola yasla, 10 karakter genişliğinde print(f"|{ad:>10}|") # Sağa yasla, 10 karakter genişliğinde print(f"|{ad:^10}|") # Ortala, 10 karakter genişliğinde # Sayıları biçimlendirme print(f"Binary: {42:b}") # İkili (binary) gösterim: 101010 print(f"Octal: {42:o}") # Sekizli (octal) gösterim: 52 print(f"Hex: {42:x}") # On altılı (hexadecimal) gösterim: 2a print(f"Hex: {42:X}") # Büyük harfle on altılı: 2A # Binlik ayırıcı kullanma print(f"Büyük sayı: {1234567:,}") # 1,234,567

2. format() Metodu

format() metodu, Python 2.6 ve sonrasında kullanılabilen esnek bir string biçimlendirme yöntemidir.

ad = "Zeynep" yas = 28 boy = 1.68 # Temel format() kullanımı print("Merhaba, benim adım {}, {} yaşındayım ve boyum {} metre.".format(ad, yas, boy)) # İndeks kullanarak sıralama print("Boyum {2} metre, {0} yaşındayım ve adım {1}.".format(yas, ad, boy)) # İsimlendirilmiş argümanlar print("Merhaba, benim adım {ad}, {yas} yaşındayım.".format(ad=ad, yas=yas)) # Sayısal değerleri biçimlendirme print("Pi sayısı: {:.2f}".format(3.14159265359)) print("Yüzde: {:.0%}".format(0.75)) # Sağa/sola yaslama ve genişlik belirtme print("|{:<10}|".format(ad)) # Sola yasla print("|{:>10}|".format(ad)) # Sağa yasla print("|{:^10}|".format(ad)) # Ortala # Sayıları biçimlendirme print("Binary: {:b}".format(42)) print("Hex: {:x}".format(42)) # Binlik ayırıcı kullanma print("Büyük sayı: {:,}".format(1234567))

3. % Operatörü (Eski Yöntem)

% operatörü, Python'un eski biçimlendirme yöntemidir. Hala kullanılmakla birlikte, yeni projelerde format() veya f-string kullanılması önerilir.

ad = "Mehmet" yas = 35 boy = 1.82 # Temel % operatörü kullanımı print("Merhaba, benim adım %s, %d yaşındayım ve boyum %.2f metre." % (ad, yas, boy)) # Yaygın format belirteçleri # %s - String # %d - Integer # %f - Float # %x - Hexadecimal # %o - Octal # %% - % işareti yazdırma # Genişlik ve hizalama print("|%-10s|" % ad) # Sola yasla print("|%10s|" % ad) # Sağa yasla # Sayısal değerleri biçimlendirme print("Pi: %.2f" % 3.14159265359) # 3.14

Dosya İşlemleri

Python'da dosya işlemleri, open() fonksiyonu kullanılarak gerçekleştirilir. Bu fonksiyon bir dosya nesnesi döndürür ve bu nesne üzerinden dosya okuma ve yazma işlemleri yapılabilir.

1. Dosya Açma ve Kapatma

# Dosya açma modları: # 'r' - Okuma modu (varsayılan) # 'w' - Yazma modu (Dosyayı oluşturur veya içeriğini siler) # 'a' - Ekleme modu (Dosyayı açar ve sonuna ekleme yapar) # 'x' - Özel oluşturma modu (Dosya zaten varsa hata verir) # 'b' - İkili (binary) mod # 't' - Metin modu (varsayılan) # Dosya açma dosya = open("metin.txt", "r", encoding="utf-8") # Dosyayı kullanma icerik = dosya.read() # Dosyayı kapatma dosya.close() # Alternatif olarak, with bloğu kullanma (önerilen) # Bu şekilde dosya otomatik olarak kapatılır with open("metin.txt", "r", encoding="utf-8") as dosya: icerik = dosya.read() print(icerik)

2. Dosya Okuma

# Tüm dosyayı okuma with open("metin.txt", "r", encoding="utf-8") as dosya: icerik = dosya.read() print(icerik) # Satır satır okuma with open("metin.txt", "r", encoding="utf-8") as dosya: for satir in dosya: print(satir, end="") # Satırların sonunda zaten \n olduğu için end="" kullanılır # readline() ile satır satır okuma with open("metin.txt", "r", encoding="utf-8") as dosya: satir = dosya.readline() while satir: print(satir, end="") satir = dosya.readline() # readlines() ile tüm satırları liste olarak okuma with open("metin.txt", "r", encoding="utf-8") as dosya: satirlar = dosya.readlines() for satir in satirlar: print(satir, end="")

3. Dosyaya Yazma

# Dosyaya yazma (içeriği siler) with open("yeni.txt", "w", encoding="utf-8") as dosya: dosya.write("Merhaba, Dünya!\n") dosya.write("Python ile dosya işlemleri.\n") # Dosyaya ekleme with open("yeni.txt", "a", encoding="utf-8") as dosya: dosya.write("Bu metin sonradan eklendi.\n") # Çoklu satır yazma satirlar = ["Birinci satır\n", "İkinci satır\n", "Üçüncü satır\n"] with open("satirlar.txt", "w", encoding="utf-8") as dosya: dosya.writelines(satirlar) # print() fonksiyonu ile yazma with open("cikti.txt", "w", encoding="utf-8") as dosya: print("Bu metin dosyaya yazdırılacak.", file=dosya) print("Bu da ikinci satır.", file=dosya)

4. Dosya Konumu ve Gezinme

# tell() - Dosya içindeki mevcut konumu verir # seek(offset, whence) - Dosya içinde gezinmeyi sağlar # - offset: Kaç bayt ileri/geri gidileceği # - whence: Başlangıç noktası (0: baştan, 1: mevcut konumdan, 2: sondan) with open("metin.txt", "r", encoding="utf-8") as dosya: # İlk 5 karakteri oku print(dosya.read(5)) # Mevcut konumu göster print("Mevcut konum:", dosya.tell()) # Dosyanın başına git dosya.seek(0) # Tekrar oku print(dosya.read(10)) # 5 karakter daha ileri git dosya.seek(5, 1) print(dosya.read(10))

Encode ve Decode

Python 3'te, string'ler Unicode karakterleri destekler. Dosya okuma ve yazma işlemlerinde, karakter kodlamasını (encoding) belirtmek önemlidir. Türkçe karakterleri doğru görüntülemek için genellikle "utf-8" kodlaması kullanılır.

# String'i bytes'a dönüştürme (encode) metin = "Merhaba, Dünya! Türkçe karakterler: çğıöşü" bytes_veri = metin.encode("utf-8") print(bytes_veri) # Bytes'ı string'e dönüştürme (decode) yeni_metin = bytes_veri.decode("utf-8") print(yeni_metin)
