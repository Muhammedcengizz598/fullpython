Python'a Giriş

Merhaba! bu dersimizde Python nedir? Ne işe yarar? ne yapılır? bu konuları öğrenicez.

Python Nedir?

Python, basit ve öğrenmesi kolay sözdizimi ile öne çıkan, yüksek seviyeli, genel amaçlı bir programlama dilidir. Guido van Rossum tarafından 1991 yılında geliştirilmeye başlanan Python, bugün dünyanın en popüler programlama dillerinden biridir.

Python'un en önemli özelliklerinden biri okunabilir kodlar yazılmasına olanak tanımasıdır. Diğer birçok programlama dilinden farklı olarak Python, girintileri (indentation) kod bloklarını belirlemek için kullanır, böylece kodlar daha temiz ve düzenli görünür.

Biliyor muydunuz?

Python ismini, Monty Python's Flying Circus adlı İngiliz komedi grubundan almıştır. Dilin dökümantasyonunda ve örneklerinde Monty Python referanslarını görmek mümkündür.

Neden Python Öğrenmeliyiz?

Python günümüzde çok geniş bir kullanım alanına sahiptir:

  • Veri Bilimi ve Makine Öğrenmesi: NumPy, Pandas, scikit-learn gibi kütüphanelerle veri analizi ve yapay zeka uygulamaları geliştirilebilir.
  • Web Geliştirme: Django, Flask gibi frameworkler ile güçlü web uygulamaları oluşturulabilir.
  • Otomasyon ve Scripting: Tekrarlayan görevleri otomatikleştirmek için ideal bir dildir.
  • Oyun Geliştirme: Pygame kütüphanesi ile 2D oyunlar geliştirilebilir.
  • Bilimsel Hesaplama: Akademik ve bilimsel araştırmalarda sıklıkla kullanılır.
  • Masaüstü Uygulamaları: Tkinter, PyQt gibi kütüphanelerle GUI uygulamaları oluşturulabilir.

Ayrıca Python, başlangıç seviyesindeki programcılar için öğrenmesi kolay olduğu kadar, profesyonel geliştiriciler için de güçlü araçlar sunar. Geniş bir topluluk desteği ve zengin kütüphane ekosistemi ile Python, neredeyse her türlü programlama ihtiyacı için uygun bir çözüm sunar.

Python Kurulumu

Python'u bilgisayarınıza kurmak oldukça basittir:

  1. Windows için: Python.org adresinden en son sürümü indirin ve kurulum sihirbazını takip edin. Kurulum sırasında "Add Python to PATH" seçeneğini işaretlemeyi unutmayın.
  2. macOS için: macOS genellikle Python ile birlikte gelir, ancak güncel sürüm için Homebrew kullanarak brew install python komutunu çalıştırabilirsiniz.
  3. Linux için: Çoğu Linux dağıtımında Python önceden kurulu gelir. Terminal üzerinden sudo apt-get install python3 (Ubuntu/Debian) veya sudo dnf install python3 (Fedora) komutlarını kullanabilirsiniz.

Kurulumun başarılı olup olmadığını kontrol etmek için, komut satırına (terminal veya komut istemi) aşağıdaki komutu yazarak Python sürümünü kontrol edebilirsiniz:

python --version # veya python3 --version

İlk Python Programı

Gelenek olduğu üzere, ilk programımız ekrana "Merhaba Dünya!" yazdıran bir program olacak:

print("Merhaba Dünya!")

Bu kodu çalıştırmak için:

  1. Bir metin editöründe (Notepad, VS Code, vb.) yukarıdaki kodu yazın ve merhaba.py olarak kaydedin.
  2. Komut satırında, dosyanın bulunduğu dizine gidin.
  3. python merhaba.py veya python3 merhaba.py komutunu çalıştırın.

Alternatif olarak, Python'un interaktif kabuğunu kullanabilirsiniz:

  1. Komut satırında python veya python3 yazarak Python kabuğunu başlatın.
  2. print("Merhaba Dünya!") kodunu yazıp Enter tuşuna basın.

Python'da Yorum Satırları

Python'da yorum satırı oluşturmak için diyez işareti (#) kullanılır. Yorum satırları kod çalıştırılırken işlenmez ve kodun ne yaptığını açıklamak için kullanılır.

# Bu bir yorum satırıdır print("Bu kod çalıştırılır") # Bu da satır sonunda bir yorum

Sonraki Adımlar

Python'a giriş konusunu tamamladıktan sonra öğrenmenize devam edebileceğiniz konular:

  • Python'da Veri Tipleri: String, Integer, Float, List, Dictionary, Tuple ve Set veri tiplerini öğrenin.
  • Koşullu İfadeler: if, elif, else yapıları ile karar mekanizmalarını öğrenin.
  • Döngüler: for ve while döngüleri ile tekrarlayan işlemleri nasıl yapacağınızı öğrenin.
  • Fonksiyonlar: Kendi fonksiyonlarınızı nasıl oluşturup kullanacağınızı öğrenin.
  • Modüller ve Kütüphaneler: Hazır modülleri kullanarak Python'un gücünü keşfedin.
