Python'da Hata Yakalama (Try-Except)

Bu dersimizde Python'da hata yakalama kavramı, try-except blokları ve hata türleri hakkında bilgi edineceksiniz.

Hata Yakalama Nedir?

Hata yakalama, programlarınızın çalışma sırasında karşılaşabileceği sorunları yönetme sürecidir. İyi tasarlanmış bir program, beklenmeyen durumlar karşısında çökmeden uygun şekilde tepki verebilmelidir.

Python'da try-except blokları kullanılarak hatalar yakalanabilir ve yönetilebilir. Bu mekanizma sayesinde programınız hata durumunda çökmek yerine, problemi tanımlayıp çözüm üretebilir.

# Hataya sebep olabilecek kod sayi1 = 10 sayi2 = 0 # Hata yakalama olmadan # print(sayi1 / sayi2) # ZeroDivisionError: division by zero # Hata yakalama ile try: sonuc = sayi1 / sayi2 print(sonuc) except ZeroDivisionError: print("Bir sayı sıfıra bölünemez!")

Biliyor muydunuz?

Python'da her hata (exception) bir sınıf örneğidir ve tüm hata sınıfları BaseException sınıfından türetilmiştir.

Try-Except Bloğu

Python'da try-except bloğunun temel yapısı şu şekildedir:

try: # Hata üretebilecek kod # ... except HataTipi: # Hata durumunda çalışacak kod # ...

Try-except bloğu genişletilerek birden fazla except, else ve finally blokları eklenebilir:

try: # Hata üretebilecek kod # ... except HataTipi1: # HataTipi1 oluştuğunda çalışacak kod # ... except HataTipi2: # HataTipi2 oluştuğunda çalışacak kod # ... else: # Hiçbir hata oluşmadığında çalışacak kod # ... finally: # Hata olsun veya olmasın her durumda çalışacak kod # ...

Yaygın Hata Türleri

Python'da sık karşılaşılan bazı hata türleri şunlardır:

1. SyntaxError

Python'un dilbilgisi kurallarına uymayan kodlarda ortaya çıkar. Bu hatalar, kodunuz çalıştırılmadan önce tespit edilir.

# SyntaxError örneği # if x > 5 # Parantez eksik # print("x 5'ten büyük")

2. TypeError

Uygunsuz veri tipi kullanımında ortaya çıkar.

# TypeError örneği try: sayi = 5 metin = "10" toplam = sayi + metin # int ve str toplanamaz print(toplam) except TypeError: print("Farklı tipteki verileri toplayamazsınız!")

3. NameError

Tanımlanmamış bir değişken veya fonksiyon kullanıldığında ortaya çıkar.

# NameError örneği try: print(tanimlanmamis_degisken) # Bu değişken tanımlanmadı except NameError: print("Bu değişken tanımlanmamış!")

4. IndexError

Bir dizinin geçersiz bir indeksine erişmeye çalıştığınızda ortaya çıkar.

# IndexError örneği try: liste = [1, 2, 3] print(liste[5]) # Liste sadece 3 eleman içeriyor except IndexError: print("Liste indeksi aralık dışında!")

5. KeyError

Bir sözlükte var olmayan bir anahtara erişmeye çalıştığınızda ortaya çıkar.

# KeyError örneği try: sozluk = {"ad": "Ali", "yas": 25} print(sozluk["meslek"]) # "meslek" anahtarı yok except KeyError: print("Bu anahtar sözlükte mevcut değil!")

6. ValueError

Doğru tip ancak uygun olmayan değerde bir argüman kullanıldığında ortaya çıkar.

# ValueError örneği try: sayi = int("abc") # "abc" tamsayıya dönüştürülemez except ValueError: print("Metin sayıya dönüştürülemedi!")

7. ZeroDivisionError

Bir sayıyı sıfıra bölmeye çalıştığınızda ortaya çıkar.

# ZeroDivisionError örneği try: sonuc = 10 / 0 except ZeroDivisionError: print("Sıfıra bölme hatası!")

8. FileNotFoundError

Olmayan bir dosyayı açmaya çalıştığınızda ortaya çıkar.

# FileNotFoundError örneği try: dosya = open("olmayan_dosya.txt", "r") except FileNotFoundError: print("Dosya bulunamadı!")

Birden Fazla Hata Yakalama

Aynı kod bloğu için birden fazla hata türünü yakalayabilirsiniz:

try: sayi = int(input("Bir sayı girin: ")) sonuc = 10 / sayi print("Sonuç:", sonuc) except ValueError: print("Geçerli bir sayı girmediniz!") except ZeroDivisionError: print("Sıfıra bölme hatası!")

Birden fazla hata türünü aynı blokta yakalamak da mümkündür:

try: sayi = int(input("Bir sayı girin: ")) sonuc = 10 / sayi print("Sonuç:", sonuc) except (ValueError, ZeroDivisionError): print("Bir hata oluştu: Ya geçersiz bir sayı girdiniz ya da sıfıra bölme hatası!")

Hata Mesajını Yakalama

Hata nesnesini yakalayarak daha ayrıntılı bilgi alabilirsiniz:

try: sayi = int(input("Bir sayı girin: ")) sonuc = 10 / sayi print("Sonuç:", sonuc) except Exception as hata: print(f"Bir hata oluştu: {hata}")

Else ve Finally Blokları

else bloğu, try bloğu hatasız çalıştığında işletilir. finally bloğu ise, hata olsun veya olmasın her durumda çalıştırılır.

try: dosya = open("veriler.txt", "r") icerik = dosya.read() except FileNotFoundError: print("Dosya bulunamadı!") else: print(f"Dosya içeriği: {icerik}") print("Dosya başarıyla okundu") finally: print("İşlem tamamlandı") try: dosya.close() # dosya açılamazsa hata vermemesi için try içinde except: pass

Kendi Hatalarınızı Oluşturma (raise)

Python'da raise anahtar kelimesi ile kendi hata durumlarınızı oluşturabilirsiniz:

def yas_kontrol(yas): if yas < 0: raise ValueError("Yaş negatif olamaz!") elif yas > 120: raise ValueError("Yaş çok büyük!") else: return f"Yaşınız: {yas}" try: print(yas_kontrol(-5)) except ValueError as hata: print(f"Hata: {hata}")

Özel Hata Sınıfları

Python'da kendi özel hata sınıflarınızı tanımlayabilirsiniz:

class YasHatasi(Exception): """Yaş ile ilgili hataları temsil eden sınıf""" pass class NegatifYasHatasi(YasHatasi): """Negatif yaş hatası""" pass class BuyukYasHatasi(YasHatasi): """Çok büyük yaş hatası""" pass def yas_kontrol(yas): if yas < 0: raise NegatifYasHatasi("Yaş negatif olamaz!") elif yas > 120: raise BuyukYasHatasi("Yaş çok büyük!") else: return f"Yaşınız: {yas}" try: print(yas_kontrol(150)) except NegatifYasHatasi as hata: print(f"Negatif yaş hatası: {hata}") except BuyukYasHatasi as hata: print(f"Büyük yaş hatası: {hata}") except YasHatasi as hata: print(f"Yaş hatası: {hata}")

Hata Ayıklama İpuçları

Hata ayıklarken, spesifik hata tiplerini yakalamak genel Exception sınıfını yakalamaktan daha iyidir. Bu şekilde, kodunuzun hangi tür hatalarla karşılaştığını daha net anlayabilirsiniz.

# Önerilen (spesifik hata yakalama) try: # kod except ValueError: # değer hatası işleme except FileNotFoundError: # dosya bulunamadı hatası işleme # Önerilmeyen (genel hata yakalama) try: # kod except Exception: # tüm hataları aynı şekilde işleme
