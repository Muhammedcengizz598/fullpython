Python'da If-Else Yapıları

Bu dersimizde Python'da koşullu ifadeler olan if, elif ve else yapıları nedir, nasıl kullanılır ve farklı kullanım senaryoları hakkında bilgi edineceksiniz.

Koşullu İfadeler Nedir?

Koşullu ifadeler, programların belirli koşullara göre farklı davranışlar sergilemesini sağlayan yapılardır. Bu yapılar, programınızın akışını kontrol etmenize ve farklı durumlara göre farklı kod bloklarının çalıştırılmasına olanak tanır.

Python'da koşullu ifadeler if, elif ve else anahtar kelimeleriyle oluşturulur ve bu yapılar sayesinde programınız karar verme yeteneği kazanır.

# Koşullu ifade örneği yas = 18 if yas >= 18: print("Reşitsiniz.") else: print("Reşit değilsiniz.") # Çıktı: "Reşitsiniz."

Biliyor muydunuz?

Python'da diğer birçok programlama dilinden farklı olarak, koşullu ifadelerde süslü parantez { } kullanılmaz. Bunun yerine girintileme (indentation) kullanılır ve kod blokları iki nokta üst üste (:) işaretinden sonra girintili satırlarla belirtilir.

Python'da Koşullu İfade Yapıları

Python'da üç temel koşullu ifade yapısı vardır:

1. if İfadesi

if ifadesi, belirtilen koşul doğru olduğunda çalıştırılacak kod bloğunu tanımlar.

# Basit if ifadesi sayi = 10 if sayi > 0: print("Sayı pozitiftir.") # Çıktı: "Sayı pozitiftir."

if ifadesinin temel sözdizimi aşağıdaki gibidir:

if koşul: # Koşul doğru ise bu blok çalışır # İstediğiniz kadar satır kod ekleyebilirsiniz

2. if-else İfadesi

if-else yapısı, bir koşul doğru olduğunda bir kod bloğunu, yanlış olduğunda ise başka bir kod bloğunu çalıştırmanıza olanak tanır.

# if-else ifadesi sayi = -5 if sayi > 0: print("Sayı pozitiftir.") else: print("Sayı pozitif değildir.") # Çıktı: "Sayı pozitif değildir."

if-else ifadesinin temel sözdizimi:

if koşul: # Koşul doğru ise bu blok çalışır else: # Koşul yanlış ise bu blok çalışır

3. if-elif-else İfadesi

if-elif-else yapısı, birden fazla koşulu sırasıyla kontrol etmenize ve bu koşullara göre farklı kod bloklarının çalıştırılmasına olanak tanır.

# if-elif-else ifadesi sayi = 0 if sayi > 0: print("Sayı pozitiftir.") elif sayi < 0: print("Sayı negatiftir.") else: print("Sayı sıfırdır.") # Çıktı: "Sayı sıfırdır."

if-elif-else ifadesinin temel sözdizimi:

if koşul1: # Koşul1 doğru ise bu blok çalışır elif koşul2: # Koşul1 yanlış ve koşul2 doğru ise bu blok çalışır elif koşul3: # Koşul1 ve koşul2 yanlış, koşul3 doğru ise bu blok çalışır else: # Hiçbir koşul doğru değilse bu blok çalışır

elif Kullanımı

elif, "else if" ifadesinin kısaltmasıdır ve istediğiniz kadar elif bloğu ekleyebilirsiniz. Python, koşulları yukarıdan aşağıya doğru kontrol eder ve ilk doğru koşulun bloğunu çalıştırır.

# Birden fazla elif kullanımı puan = 85 if puan >= 90: print("AA") elif puan >= 80: print("BA") elif puan >= 70: print("BB") elif puan >= 60: print("CB") elif puan >= 50: print("CC") else: print("FF") # Çıktı: "BA"

Koşul Operatörleri

Koşullu ifadelerde kullanılabilecek çeşitli operatörler vardır:

  • == (Eşittir): İki değerin eşit olup olmadığını kontrol eder
  • != (Eşit Değildir): İki değerin farklı olup olmadığını kontrol eder
  • < (Küçüktür): Sol değerin sağ değerden küçük olup olmadığını kontrol eder
  • > (Büyüktür): Sol değerin sağ değerden büyük olup olmadığını kontrol eder
  • <= (Küçük veya Eşittir): Sol değerin sağ değerden küçük veya eşit olup olmadığını kontrol eder
  • >= (Büyük veya Eşittir): Sol değerin sağ değerden büyük veya eşit olup olmadığını kontrol eder
# Karşılaştırma operatörleri örneği a = 5 b = 10 if a == b: print("a, b'ye eşittir.") elif a != b: print("a, b'ye eşit değildir.") if a < b: print("a, b'den küçüktür.") elif a > b: print("a, b'den büyüktür.") # Çıktı: # "a, b'ye eşit değildir." # "a, b'den küçüktür."

Mantıksal Operatörler

Python'da, birden fazla koşulu birleştirmek için mantıksal operatörler kullanılır:

  • and: Her iki koşul da doğru olmalıdır
  • or: En az bir koşul doğru olmalıdır
  • not: Koşulun tersini alır
# Mantıksal operatörler örneği yas = 25 mezuniyet = "üniversite" # and operatörü if yas >= 18 and mezuniyet == "üniversite": print("İşe alım kriterlerine uygunsunuz.") # or operatörü if yas >= 30 or mezuniyet == "üniversite": print("Terfi için değerlendirilebilirsiniz.") # not operatörü if not (yas < 18): print("Reşitsiniz.") # Çıktı: # "İşe alım kriterlerine uygunsunuz." # "Terfi için değerlendirilebilirsiniz." # "Reşitsiniz."

İç İçe (Nested) if İfadeleri

Python'da bir if ifadesi içinde başka bir if ifadesi kullanabilirsiniz. Bu, iç içe (nested) koşullu ifadeler olarak adlandırılır.

# İç içe if ifadeleri yas = 25 gelir = 5000 if yas >= 18: print("Reşitsiniz.") if gelir >= 3000: print("Kredi başvurusu yapabilirsiniz.") else: print("Gelir düzeyiniz kredi başvurusu için yetersiz.") else: print("Reşit olmadığınız için kredi başvurusu yapamazsınız.")

Tek Satırlık if İfadeleri

Python'da basit if ifadelerini tek satırda yazabilirsiniz. Bu, özellikle kısa ve basit koşullu işlemler için kullanışlıdır.

# Tek satırlık if ifadesi sayi = 10 if sayi > 0: print("Sayı pozitiftir.") # if-else tek satırda yas = 20 print("Reşitsiniz.") if yas >= 18 else print("Reşit değilsiniz.")

Koşullu İfadelerde pass İfadesi

Python'da bazen bir koşul için hiçbir şey yapmak istemeyebilirsiniz. Bu durumda pass ifadesi kullanılır.

# pass ifadesi örneği sayi = 5 if sayi > 0: pass # Burada şimdilik bir şey yapmak istemiyoruz else: print("Sayı negatif veya sıfır.")

Ternary Operatör (Koşul Operatörü)

Python'da tek satırda if-else mantığı uygulayabileceğiniz bir koşul operatörü bulunur. Bu, bazı dillerdeki ternary operatörün Python'daki karşılığıdır.

# Ternary operatör örneği yas = 20 durum = "Reşit" if yas >= 18 else "Reşit değil" print(durum) # Çıktı: "Reşit" # Değer atama örneği a = 10 b = 20 buyuk_olan = a if a > b else b print(f"Büyük olan sayı: {buyuk_olan}") # Çıktı: "Büyük olan sayı: 20"

Koşul Olarak Değerlendirilen İfadeler

Python'da, bir koşulun doğru olarak değerlendirilmesi için mutlaka bir karşılaştırma operatörü kullanmanız gerekmez. Aşağıdaki değerler otomatik olarak False olarak değerlendirilir:

  • None
  • False
  • 0 (sıfır) - herhangi bir sayısal türde (0, 0.0, 0j)
  • Boş diziler - '', [], (), {}, set()

Diğer tüm değerler True olarak değerlendirilir.

# Koşul olarak değerlendirilen değerler liste = [] if not liste: print("Liste boş.") sayi = 0 if not sayi: print("Sayı sıfır.") metin = "Python" if metin: print("Metin boş değil.") # Çıktı: # "Liste boş." # "Sayı sıfır." # "Metin boş değil."

İpucu

Bir değişkenin boş olup olmadığını kontrol etmek için doğrudan değişkeni koşul olarak kullanabilirsiniz. Örneğin, `if liste:` ifadesi listenin boş olmadığını kontrol eder.

in ve not in Operatörleri

Koşullu ifadelerde, bir değerin bir koleksiyon içinde olup olmadığını kontrol etmek için in ve not in operatörleri kullanılabilir.

# in ve not in operatörleri meyveler = ["elma", "armut", "muz", "portakal"] if "elma" in meyveler: print("Elma, meyveler listesinde var.") if "kiraz" not in meyveler: print("Kiraz, meyveler listesinde yok.") # String içinde kontrol isim = "Python" if "P" in isim: print("P harfi isimde var.") # Sözlük içinde kontrol (anahtarlar için) kisi = {"ad": "Ahmet", "yas": 25} if "ad" in kisi: print("Kişinin adı:", kisi["ad"])

is ve is not Operatörleri

is ve is not operatörleri, iki nesnenin aynı nesne olup olmadığını (bellekte aynı yeri işaret edip etmediğini) kontrol eder.

# is ve is not operatörleri a = [1, 2, 3] b = [1, 2, 3] c = a print(a == b) # True (değerleri aynı) print(a is b) # False (farklı nesneler) print(a is c) # True (aynı nesne) print(a is not b) # True (farklı nesneler) # None kontrolü için ideal x = None if x is None: print("x değeri None'dır.")

Dikkat

None, True, False gibi değerleri kontrol ederken == yerine is operatörünü kullanmak daha doğrudur. Örneğin, `if x is None:` ifadesi `if x == None:` ifadesinden daha tercih edilir.
