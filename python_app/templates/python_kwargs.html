Python'da *args ve **kwargs

Bu dersimizde Python'da *args ve **kwargs nedir, nasıl kullanılır ve fonksiyonları daha esnek hale getirmek için nasıl faydalanılır öğreneceksiniz.

*args ve **kwargs Nedir?

Python'da *args ve **kwargs, fonksiyonlara değişken sayıda argüman geçmemizi sağlayan özel parametrelerdir. Bu yapılar, fonksiyonları daha esnek ve yeniden kullanılabilir hale getirmek için kullanılır.

Bu yapılar sayesinde kaç tane argüman alacağını önceden bilmediğimiz fonksiyonlar oluşturabiliriz.

# Normal bir fonksiyon - sabit sayıda parametre def topla(a, b): return a + b # *args ile değişken sayıda argüman alabilen fonksiyon def topla_hepsi(*args): toplam = 0 for sayi in args: toplam += sayi return toplam # Normal fonksiyon çağrısı print(topla(5, 3)) # 8 # *args ile fonksiyon çağrısı print(topla_hepsi(1, 2, 3, 4, 5)) # 15 print(topla_hepsi(10, 20)) # 30 print(topla_hepsi(7)) # 7 print(topla_hepsi()) # 0

Biliyor muydunuz?

*args ve **kwargs isimleri sadece bir gelenektir. Aslında * ve ** operatörleri önemlidir. *meyveler ve **ozellikler gibi isimler de kullanılabilir, ama okunabilirlik açısından yaygın kullanım *args ve **kwargs şeklindedir.

*args Kullanımı

*args parametresi, fonksiyona istediğimiz sayıda argüman geçmemizi sağlar ve bu argümanlar bir tuple (demet) olarak fonksiyona aktarılır.

def fonksiyon_bilgisi(fonksiyon_adi, *args): print(f"{fonksiyon_adi} fonksiyonu {len(args)} argüman aldı:") for i, arg in enumerate(args): print(f"Argüman {i+1}: {arg}") fonksiyon_bilgisi("test", "Python", 42, True, [1, 2, 3]) # Çıktı: # test fonksiyonu 4 argüman aldı: # Argüman 1: Python # Argüman 2: 42 # Argüman 3: True # Argüman 4: [1, 2, 3]

*args kullanırken dikkat edilmesi gereken noktalar:

  • *args bir tuple olarak işlenir
  • Fonksiyon içinde args değişkeni üzerinden erişim sağlanır (* olmadan)
  • Normal parametreler ile birlikte kullanılabilir, ama normal parametreler *args'tan önce tanımlanmalıdır
# Normal parametreler ile *args kullanımı def kisi_bilgisi(isim, yas, *hobiler): print(f"İsim: {isim}") print(f"Yaş: {yas}") print("Hobiler:") for hobi in hobiler: print(f"- {hobi}") kisi_bilgisi("Ali", 25, "Yüzme", "Kitap okuma", "Satranç") # Çıktı: # İsim: Ali # Yaş: 25 # Hobiler: # - Yüzme # - Kitap okuma # - Satranç

**kwargs Kullanımı

**kwargs parametresi ise fonksiyona anahtar-değer çiftleri olarak argümanlar geçmemizi sağlar ve bu argümanlar bir dictionary (sözlük) olarak fonksiyona aktarılır.

def profil_olustur(**kwargs): print("Profil Bilgileri:") for anahtar, deger in kwargs.items(): print(f"{anahtar}: {deger}") profil_olustur(isim="Ayşe", yas=30, meslek="Mühendis", sehir="İstanbul") # Çıktı: # Profil Bilgileri: # isim: Ayşe # yas: 30 # meslek: Mühendis # sehir: İstanbul

**kwargs kullanırken dikkat edilmesi gereken noktalar:

  • **kwargs bir dictionary (sözlük) olarak işlenir
  • Fonksiyon içinde kwargs değişkeni üzerinden erişim sağlanır (** olmadan)
  • Normal parametreler ve *args ile birlikte kullanılabilir, ama **kwargs en sonda olmalıdır
# Normal parametreler, *args ve **kwargs birlikte kullanımı def urun_bilgisi(urun_adi, fiyat, *ozellikler, **detaylar): print(f"Ürün: {urun_adi}") print(f"Fiyat: {fiyat} TL") print("Özellikler:") for ozellik in ozellikler: print(f"- {ozellik}") print("Detaylar:") for anahtar, deger in detaylar.items(): print(f"{anahtar}: {deger}") urun_bilgisi("Laptop", 15000, "16GB RAM", "512GB SSD", "15.6 inç", marka="Asus", renk="Siyah", garanti_suresi="2 yıl") # Çıktı: # Ürün: Laptop # Fiyat: 15000 TL # Özellikler: # - 16GB RAM # - 512GB SSD # - 15.6 inç # Detaylar: # marka: Asus # renk: Siyah # garanti_suresi: 2 yıl

Liste ve Sözlükleri Açma (Unpacking)

* ve ** operatörleri sadece fonksiyon tanımlamada değil, fonksiyon çağrısında liste ve sözlükleri açmak (unpacking) için de kullanılabilir.

# Liste açma (liste -> argümanlar) def topla(a, b, c): return a + b + c sayilar = [1, 2, 3] print(topla(*sayilar)) # 6 # Sözlük açma (sözlük -> isimli argümanlar) def kisi_bilgileri(isim, yas, sehir): return f"{isim}, {yas} yaşında, {sehir}'de yaşıyor" bilgiler = {"isim": "Mehmet", "yas": 35, "sehir": "Ankara"} print(kisi_bilgileri(**bilgiler)) # Mehmet, 35 yaşında, Ankara'de yaşıyor

Fonksiyon Çağrılarında *args ve **kwargs Kullanımı

*args ve **kwargs yapısı, bir fonksiyonun aldığı tüm argümanları başka bir fonksiyona doğrudan iletmek için de kullanılabilir.

# Dekoratör fonksiyon örneği def zamanlayici(func): import time def wrapper(*args, **kwargs): baslangic = time.time() sonuc = func(*args, **kwargs) # Orijinal fonksiyonu çağır bitis = time.time() print(f"Fonksiyon çalışma süresi: {bitis - baslangic:.5f} saniye") return sonuc return wrapper @zamanlayici def topla(*args): import time time.sleep(1) # 1 saniye bekle return sum(args) print(topla(1, 2, 3, 4, 5)) # 15 ve çalışma süresini yazdırır

En Yaygın Kullanım Alanları

*args ve **kwargs en çok şu durumlarda kullanılır:

  • Dekoratör fonksiyonlarda orijinal fonksiyonun argümanlarını iletmek için
  • Fonksiyonları genişletilebilir ve esnek hale getirmek için
  • Kalıtım yapısında üst sınıf metotlarının parametrelerini alt sınıflara geçirmek için
# Kalıtımda üst sınıf metotları için örnek class Sekil: def __init__(self, renk="Beyaz", **kwargs): self.renk = renk super().__init__(**kwargs) # Üst sınıfa tüm argümanları iletir class Dikdortgen(Sekil): def __init__(self, genislik, yukseklik, **kwargs): self.genislik = genislik self.yukseklik = yukseklik super().__init__(**kwargs) # Şekil sınıfına geriye kalan argümanları iletir

Yapılabilecek Projeler

Python'da *args ve **kwargs konusunda pratik yapmak için aşağıdaki projeleri deneyebilirsiniz:

Gelişmiş Hesap Makinesi

*args ve **kwargs kullanarak çeşitli matematiksel işlemleri (toplama, çıkarma, çarpma, bölme) yapabilen ve farklı seçenekler (ondalık basamak, yuvarlama) sunabilen bir hesap makinesi fonksiyonu yazın.

Başlangıç

Form Doğrulayıcı

**kwargs kullanarak formlardan gelen verileri doğrulayan (e-posta formatı, şifre gücü, yaş kontrolü vb.) bir fonksiyon geliştirin.

Başlangıç

Veri Analiz Aracı

*args ile bir liste verilerini alıp, **kwargs ile belirtilen analizleri (ortalama, medyan, standart sapma, min, max vb.) gerçekleştiren bir fonksiyon yazın.

Başlangıç

Gelişmiş Loglama Sistemi

*args ve **kwargs kullanan bir dekoratör fonksiyon yazın. Bu dekoratör, herhangi bir fonksiyonun girdi parametrelerini, çıktılarını ve çalışma süresini loglasın.

Kendinizi Test Edin

1. Python'da *args ne tür bir veri yapısı olarak fonksiyon içinde işlenir?

A
Liste (List)
B
Demet (Tuple)
C
Sözlük (Dictionary)
D
Küme (Set)
Doğru! *args, fonksiyon içinde bir demet (tuple) olarak işlenir.
Yanlış. *args, fonksiyon içinde bir demet (tuple) olarak işlenir.

2. Aşağıdaki Python kodunun çıktısı ne olur?

def fonksiyon(x, *args):
    print(x)
    print(args)

10, 20, 30, 40
B
[10, 20, 30, 40]

10
(20, 30, 40)
D

Doğru! İlk argüman x'e atanır (10), geri kalan argümanlar args demetine atanır (20, 30, 40).
Yanlış. İlk argüman x'e atanır (10), geri kalan argümanlar args demetine atanır (20, 30, 40).

3. **kwargs içinde anahtar-değer çiftlerine nasıl erişilir?

A
kwargs[0], kwargs[1] gibi indekslerle
B
kwargs.items(), kwargs["anahtar"] gibi sözlük metodlarıyla
C
for x in kwargs: gibi doğrudan değerleri iterasyonla
D
kwargs.get_item("anahtar") şeklinde
Doğru! **kwargs bir sözlük olarak işlendiği için, sözlük metodları (items(), keys(), values()) kullanılabilir ve anahtar adlarıyla değerlere erişilebilir.
Yanlış. **kwargs bir sözlük olarak işlendiği için, sözlük metodları (items(), keys(), values()) kullanılabilir ve anahtar adlarıyla değerlere erişilebilir.

def fonksiyon(*args, x, **kwargs):

Kod hatasızdır
Doğru! *args parametresinden sonra normal parametre tanımlanamaz. Bunun yerine, x parametresi *args'tan önce tanımlanmalıdır: def fonksiyon(x, *args, **kwargs)
Yanlış. *args parametresinden sonra normal parametre tanımlanamaz. Bunun yerine, x parametresi *args'tan önce tanımlanmalıdır: def fonksiyon(x, *args, **kwargs)

7. *args parametresi için hangisi doğrudur?

Sonraki Adımlar

Python *args ve **kwargs konusunu tamamladıktan sonra öğrenmenize devam edebileceğiniz konular:

  • Dekoratörler: Fonksiyonların davranışlarını değiştiren güçlü bir Python özelliği.
  • Lambda Fonksiyonları: Tek satırda tanımlanan anonim fonksiyonlar.
  • İleri Seviye Fonksiyonlar: İç içe fonksiyonlar ve kapsamlar (scopes).
  • Nesne Yönelimli Programlama: Sınıflar, nesneler ve inheritance kavramları.
  • Metod Overriding: Üst sınıfın metodlarını alt sınıfta yeniden tanımlama.

Başarı İpucu

Python'da *args ve **kwargs konusunu öğrenirken, farklı kombinasyonlarda kullanarak pratik yapın. Özellikle fonksiyon dekoratörleri ve miras (inheritance) senaryolarında *args ve **kwargs kullanımı çok faydalıdır!
