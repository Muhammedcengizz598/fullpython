Python'da Listeler

Bu dersimizde Python'da listeler nedir, nasıl oluşturulur ve liste metodları hakkında detaylı bilgi edineceksiniz.

Listeler Nedir?

Listeler, Python'un en çok kullanılan veri yapılarından biridir. Listeler, farklı veri tiplerini bir arada tutabilen, sıralı ve değiştirilebilir (mutable) koleksiyonlardır.

Python'da listeler köşeli parantezler [] içinde tanımlanır ve elemanlar virgülle ayrılır.

# Liste oluşturma bos_liste = [] sayilar = [1, 2, 3, 4, 5] meyveler = ["elma", "armut", "muz", "çilek"] karisik = [1, "elma", 3.14, True, [1, 2, 3]] # Listeyi ekrana yazdırma print("Boş liste:", bos_liste) print("Sayılar:", sayilar) print("Meyveler:", meyveler) print("Karışık liste:", karisik)

Biliyor muydunuz?

Python listeleri diğer programlama dillerindeki dizilere (array) benzer, ancak onlardan daha esnektir. Listeler farklı veri tiplerini bir arada tutabilir ve boyutları dinamik olarak değişebilir.

Liste Elemanlarına Erişim

Liste elemanlarına indeks numarası ile erişebilirsiniz. Python'da indeks numaraları 0'dan başlar.

meyveler = ["elma", "armut", "muz", "çilek", "kivi"] # İndeks ile elemanlara erişim print("İlk meyve:", meyveler[0]) # elma print("İkinci meyve:", meyveler[1]) # armut print("Son meyve:", meyveler[4]) # kivi # Negatif indeks kullanımı (sondan erişim) print("Son meyve:", meyveler[-1]) # kivi print("Sondan ikinci meyve:", meyveler[-2]) # çilek # Hata durumu: Olmayan indekse erişmeye çalışmak # print(meyveler[10]) # IndexError: list index out of range

Liste Dilimleme (Slicing)

Liste dilimleme, bir listenin belirli bir bölümünü alma işlemidir. Sözdizimi şu şekildedir: liste[başlangıç:bitiş:adım]

meyveler = ["elma", "armut", "muz", "çilek", "kivi", "ananas", "üzüm"] # Liste dilimleme print("İlk iki meyve:", meyveler[0:2]) # ["elma", "armut"] print("İlk üç meyve:", meyveler[:3]) # ["elma", "armut", "muz"] print("Son üç meyve:", meyveler[-3:]) # ["kivi", "ananas", "üzüm"] print("İlk indeksten itibaren iki atlayarak:", meyveler[::2]) # ["elma", "muz", "kivi", "üzüm"] print("Listeyi ters çevirme:", meyveler[::-1]) # ["üzüm", "ananas", "kivi", "çilek", "muz", "armut", "elma"]

Liste Metodları

Python listeleri, üzerinde çeşitli işlemler yapmamızı sağlayan birçok metoda sahiptir.

1. Eleman Ekleme ve Çıkarma

meyveler = ["elma", "armut", "muz"] print("Orijinal liste:", meyveler) # Listenin sonuna eleman ekleme: append() meyveler.append("çilek") print("append() sonrası:", meyveler) # ["elma", "armut", "muz", "çilek"] # Belirli bir indekse eleman ekleme: insert() meyveler.insert(1, "portakal") print("insert() sonrası:", meyveler) # ["elma", "portakal", "armut", "muz", "çilek"] # Listenin sonundan eleman çıkarma: pop() cikarilan = meyveler.pop() print("pop() sonrası:", meyveler) # ["elma", "portakal", "armut", "muz"] print("Çıkarılan eleman:", cikarilan) # çilek # Belirli bir indeksten eleman çıkarma: pop(indeks) cikarilan = meyveler.pop(1) print("pop(1) sonrası:", meyveler) # ["elma", "armut", "muz"] print("Çıkarılan eleman:", cikarilan) # portakal # Belirli bir elemanı silme: remove() meyveler.remove("armut") print("remove() sonrası:", meyveler) # ["elma", "muz"] # Listenin tamamını temizleme: clear() meyveler.clear() print("clear() sonrası:", meyveler) # []

2. Liste Birleştirme ve Çoğaltma

liste1 = [1, 2, 3] liste2 = [4, 5, 6] # İki listeyi birleştirme birlesik = liste1 + liste2 print("Birleştirilmiş liste:", birlesik) # [1, 2, 3, 4, 5, 6] # extend() metoduyla listeye başka bir liste ekleme liste1.extend(liste2) print("extend() sonrası liste1:", liste1) # [1, 2, 3, 4, 5, 6] # Listeyi çoğaltma coklu = [0] * 5 print("Çoğaltılmış liste:", coklu) # [0, 0, 0, 0, 0] # Liste içinde liste çoğaltma (dikkat: sığ kopyalama yapar) ic_liste = [[0]] * 3 print("İç içe liste:", ic_liste) # [[0], [0], [0]] # İlk iç listeyi değiştirdiğimizde tüm kopyalar etkilenir ic_liste[0].append(1) print("Değişiklik sonrası:", ic_liste) # [[0, 1], [0, 1], [0, 1]]

3. Sıralama ve Ters Çevirme

sayilar = [5, 2, 8, 1, 9, 3] meyveler = ["muz", "elma", "çilek", "armut"] # Liste sıralama: sort() sayilar.sort() print("Sıralanmış sayılar:", sayilar) # [1, 2, 3, 5, 8, 9] meyveler.sort() print("Sıralanmış meyveler:", meyveler) # ['armut', 'elma', 'muz', 'çilek'] # Ters sıralama sayilar.sort(reverse=True) print("Ters sıralanmış sayılar:", sayilar) # [9, 8, 5, 3, 2, 1] # Sıralanmış yeni bir liste oluşturma: sorted() sayilar_orijinal = [5, 2, 8, 1, 9, 3] sayilar_siralanmis = sorted(sayilar_orijinal) print("Orijinal liste:", sayilar_orijinal) # [5, 2, 8, 1, 9, 3] print("Sıralanmış yeni liste:", sayilar_siralanmis) # [1, 2, 3, 5, 8, 9] # Listeyi ters çevirme: reverse() meyveler.reverse() print("Ters çevrilmiş liste:", meyveler) # ['çilek', 'muz', 'elma', 'armut']

4. Liste Arama ve Sayma

sayilar = [1, 2, 3, 2, 4, 2, 5] meyveler = ["elma", "armut", "muz", "elma", "çilek"] # Eleman sayma: count() print("2 sayısının adedi:", sayilar.count(2)) # 3 print("elma adedi:", meyveler.count("elma")) # 2 # Eleman indeksini bulma: index() print("3 sayısının indeksi:", sayilar.index(3)) # 2 print("muz'un indeksi:", meyveler.index("muz")) # 2 # Belirli bir aralıkta arama print("2'nin ilk bulunduğu indeks:", sayilar.index(2)) # 1 print("2'nin 2. indeksten sonra bulunduğu indeks:", sayilar.index(2, 2)) # 3 # in operatörü ile eleman kontrolü print("5 listede var mı?", 5 in sayilar) # True print("kiraz listede var mı?", "kiraz" in meyveler) # False

5. Liste Kopyalama

# Liste kopyalamanın farklı yolları liste1 = [1, 2, 3, 4, 5] # Basit atama - Bu şekilde referans kopyalanır, yeni liste oluşmaz! liste2 = liste1 liste2[0] = 100 print("liste1:", liste1) # [100, 2, 3, 4, 5] print("liste2:", liste2) # [100, 2, 3, 4, 5] # Sığ kopyalama (shallow copy) liste1 = [1, 2, 3, 4, 5] liste2 = liste1.copy() # veya liste2 = liste1[:] veya liste2 = list(liste1) liste2[0] = 100 print("liste1:", liste1) # [1, 2, 3, 4, 5] print("liste2:", liste2) # [100, 2, 3, 4, 5] # İç içe listelerde sığ kopyalama yeterli olmaz ic_ice = [[1, 2], [3, 4]] kopya = ic_ice.copy() kopya[0][0] = 100 print("ic_ice:", ic_ice) # [[100, 2], [3, 4]] print("kopya:", kopya) # [[100, 2], [3, 4]] # Derin kopyalama (deep copy) import copy ic_ice = [[1, 2], [3, 4]] derin_kopya = copy.deepcopy(ic_ice) derin_kopya[0][0] = 100 print("ic_ice:", ic_ice) # [[1, 2], [3, 4]] print("derin_kopya:", derin_kopya) # [[100, 2], [3, 4]]

Liste Fonksiyonları

Python'da listelerle çalışırken kullanabileceğiniz önemli fonksiyonlar:

sayilar = [5, 2, 8, 1, 9, 3] # Listenin uzunluğu: len() print("Liste uzunluğu:", len(sayilar)) # 6 # En küçük ve en büyük değerler: min(), max() print("En küçük değer:", min(sayilar)) # 1 print("En büyük değer:", max(sayilar)) # 9 # Toplam: sum() print("Elemanların toplamı:", sum(sayilar)) # 28 # Eleman sayısı: count() print("3 sayısının adedi:", sayilar.count(3)) # 1 # Listeyi string'e dönüştürme: join() meyveler = ["elma", "armut", "muz"] metin = ", ".join(meyveler) print("Metin:", metin) # "elma, armut, muz" # String'i listeye dönüştürme: split() metin = "elma,armut,muz" liste = metin.split(",") print("Liste:", liste) # ["elma", "armut", "muz"]

Liste Üretme (List Comprehension)

Liste üretme (list comprehension), tek satırda liste oluşturmak için kullanılan kısa ve etkili bir yöntemdir.

# For döngüsü ile liste oluşturma kareler = [] for i in range(1, 6): kareler.append(i ** 2) print("For döngüsü ile:", kareler) # [1, 4, 9, 16, 25] # Liste üretme (list comprehension) ile aynı işlem kareler = [i ** 2 for i in range(1, 6)] print("List comprehension ile:", kareler) # [1, 4, 9, 16, 25] # Koşullu liste üretme cift_sayilar = [i for i in range(1, 11) if i % 2 == 0] print("Çift sayılar:", cift_sayilar) # [2, 4, 6, 8, 10] # İç içe for döngüleri ile liste üretme koordinatlar = [(x, y) for x in range(1, 4) for y in range(1, 3)] print("Koordinatlar:", koordinatlar) # [(1, 1), (1, 2), (2, 1), (2, 2), (3, 1), (3, 2)] # if-else ile liste üretme sonuclar = ["çift" if x % 2 == 0 else "tek" for x in range(1, 6)] print("Sonuçlar:", sonuclar) # ['tek', 'çift', 'tek', 'çift', 'tek']

2B Listeler (Matrisler)

Python'da iki boyutlu listeler (matrisler), listeler içinde liste şeklinde oluşturulabilir.

# 2B liste (matris) oluşturma matris = [ [1, 2, 3], [4, 5, 6], [7, 8, 9] ] # Matris elemanlarına erişim print("İlk satır:", matris[0]) # [1, 2, 3] print("İkinci satır, üçüncü sütun:", matris[1][2]) # 6 # Matris üzerinde döngü ile gezinme for satir in matris: for eleman in satir: print(eleman, end=" ") print() # Yeni satıra geç # List comprehension ile matris işlemleri # Her elemanın karesini alma kare_matris = [[eleman ** 2 for eleman in satir] for satir in matris] print("Kareleri alınmış matris:", kare_matris) # [[1, 4, 9], [16, 25, 36], [49, 64, 81]]

Kendinizi Test Edin

Python'da listeler konusunu ne kadar anladığınızı test etmek için aşağıdaki soruları cevaplayın:

1. Aşağıdaki liste tanımlamalarından hangisi doğrudur?

A
liste = (1, 2, 3, 4)
B
liste = [1, 2, 3, 4]
C
liste = {1, 2, 3, 4}
D
liste = <1, 2, 3, 4>
Doğru! Python'da listeler köşeli parantez [] ile tanımlanır.
Yanlış. Python'da listeler köşeli parantezler [] kullanılarak tanımlanır. A seçeneği tuple, C seçeneği set tanımlamasıdır.

2. Aşağıdaki Python kodunun çıktısı ne olur?

liste = [1, 2, 3, 4, 5]
print(liste[1:4])
A
[1, 2, 3, 4]
B
[2, 3, 4]
C
[1, 2, 3]
D
[2, 3, 4, 5]
Doğru! liste[1:4] ifadesi 1. indeksten başlayıp 4. indekse kadar (4 dahil değil) olan elemanları verir: [2, 3, 4]
Yanlış. liste[1:4] ifadesi 1. indeksten başlayıp 4. indekse kadar (4 dahil değil) olan elemanları verir: [2, 3, 4]

3. Liste metodlarıyla ilgili aşağıdakilerden hangisi yanlıştır?

A
append() metodu listenin sonuna eleman ekler.
B
remove() metodu belirtilen elemanı listeden çıkarır.
C
sort() metodu yeni bir sıralanmış liste döndürür.
D
clear() metodu listenin tüm elemanlarını siler.
Doğru! sort() metodu yeni bir liste döndürmez, mevcut listeyi yerinde (in-place) sıralar. Yeni bir sıralanmış liste elde etmek için sorted() fonksiyonu kullanılır.
Yanlış. sort() metodu yeni bir liste döndürmez, mevcut listeyi yerinde (in-place) sıralar. Yeni bir sıralanmış liste elde etmek için sorted() fonksiyonu kullanılır.

Yanlış. liste2 = liste1 ifadesi, liste1'in referansını liste2'ye kopyalar. Bu yüzden liste1'de yapılan değişiklikler liste2'yi de etkiler.

5. Aşağıdaki Python listesinin çıktısı nedir?

liste = [1, 2, 3, 4, 5]
print(liste[1:4])

[1, 2, 3, 4]
B
[2, 3, 4]

[1, 2, 3]
D
[2, 3, 4, 5]
Doğru! Python'da dilimlemede (slicing) başlangıç indeksi dahil, bitiş indeksi hariç alınır. liste[1:4] ifadesi 1. indeksten (2. eleman) başlayıp 4. indekse (5. eleman) kadar olan elemanları içerir, ancak 4. indeks dahil değildir.
Yanlış. Python'da dilimlemede (slicing) başlangıç indeksi dahil, bitiş indeksi hariç alınır. liste[1:4] ifadesi 1. indeksten (2. eleman) başlayıp 4. indekse (5. eleman) kadar olan elemanları içerir, ancak 4. indeks dahil değildir.

6. Aşağıdaki liste metodlarından hangisi bir liste elemanını kaldırır ve değerini döndürür?

A
remove()
B
pop()
C
clear()
D
del
Doğru! pop() metodu, belirtilen indeksteki elemanı listeden kaldırır ve o elemanın değerini döndürür. Varsayılan olarak son elemanı kaldırır.
Yanlış. remove() metodu belirtilen değeri listeden kaldırır ama değer döndürmez. clear() metodu listeyi tamamen boşaltır. del ifadesi liste elemanını kaldırır ama değerini döndürmez.

7. Python'da liste ile ilgili aşağıdakilerden hangisi doğrudur?

A
Listeler değiştirilemez (immutable) veri yapılarıdır
B
Listeler sadece aynı tipteki verileri saklayabilir
C
Listeler iç içe olabilir (nested lists)
D
Listeler anahtarlarla erişilebilir
Doğru! Python'da listeler iç içe olabilir, yani bir liste başka listeleri eleman olarak içerebilir. Listeler değiştirilebilir (mutable), farklı veri tiplerini içerebilir ve indeks numaralarıyla erişilir.
Yanlış. Listeler değiştirilebilir (mutable) veri yapılarıdır, farklı tipteki verileri saklayabilirler ve indeks numaralarıyla erişilirler. Anahtarla erişim sözlüklere (dictionary) özgüdür.
