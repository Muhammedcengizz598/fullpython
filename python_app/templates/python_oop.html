Python'da Nesne Yönelimli Programlama (OOP)

Bu dersimizde Python'da Nesne Yönelimli Programlama kavramlarını, sınıflar ve nesnelerin nasıl oluşturulduğunu ve kullanıldığını öğreneceksiniz.

Nesne Yönelimli Programlama Nedir?

Nesne Yönelimli Programlama (OOP), programlama problemlerini gerçek hayattaki nesnelere benzer yapılar oluşturarak çözmeyi hedefleyen bir programlama paradigmasıdır. Bu yaklaşım, kodun yeniden kullanılabilirliğini artırır ve büyük projeleri daha yönetilebilir hale getirir.

Python, nesne yönelimli programlama için sınıflar (class) ve nesneler (object) kullanır. Bir sınıf, nesnelerin şablonu veya taslağı olarak düşünülebilir; nesneler ise bu şablondan üretilen somut örneklerdir.

# Basit bir sınıf tanımlama class Araba: # Sınıfın özellikleri (attributes) marka = "Toyota" model = "Corolla" yil = 2020 # Sınıfın metodu (method) def bilgileri_goster(self): return f"{self.yil} model {self.marka} {self.model}" # Bir nesne oluşturma benim_arabam = Araba() print(benim_arabam.bilgileri_goster()) # 2020 model Toyota Corolla

Biliyor muydunuz?

Python'da her şey bir nesnedir. Tamsayılar, dizeler, listeler, fonksiyonlar - hepsi birer nesnedir ve belirli sınıfların örnekleridir.

OOP'nin Temel Kavramları

Nesne Yönelimli Programlama dört temel kavram üzerine kurulmuştur:

  • Soyutlama (Abstraction): Gereksiz detayları gizleyip sadece gerekli özellikleri gösterme.
  • Kapsülleme (Encapsulation): Verileri ve bu veriler üzerinde çalışan metotları tek bir birim içinde tutma.
  • Kalıtım (Inheritance): Bir sınıfın başka bir sınıfın özellik ve davranışlarını devralması.
  • Çok Biçimlilik (Polymorphism): Aynı arayüzün farklı sınıflar için farklı davranışlar sergilemesi.

Sınıflar ve Nesneler

Python'da sınıf tanımlamak için class anahtar kelimesi kullanılır:

class Ogrenci: # Constructor metodu (Başlatıcı) def __init__(self, ad, soyad, numara): self.ad = ad self.soyad = soyad self.numara = numara # Sınıf metodu def tam_ad(self): return f"{self.ad} {self.soyad}" # Sınıf metodu def derece_al(self, ders_notu): if ders_notu >= 90: return "AA" elif ders_notu >= 80: return "BA" elif ders_notu >= 70: return "BB" elif ders_notu >= 60: return "CB" elif ders_notu >= 50: return "CC" else: return "FF" # Öğrenci nesnesi oluşturma ogrenci1 = Ogrenci("Ayşe", "Yılmaz", "12345") print(ogrenci1.tam_ad()) # Ayşe Yılmaz print(ogrenci1.derece_al(85)) # BA

__init__ Metodu

__init__ metodu, sınıfın yapıcı (constructor) metodudur. Bir nesne oluşturulduğunda otomatik olarak çağrılır ve nesnenin başlangıç durumunu ayarlar.

self parametresi, sınıfın kendisini temsil eder ve sınıf içinde sınıfın özelliklerine ve metotlarına erişim sağlar.

class Dikdortgen: def __init__(self, uzunluk, genislik): self.uzunluk = uzunluk self.genislik = genislik def alan_hesapla(self): return self.uzunluk * self.genislik def cevre_hesapla(self): return 2 * (self.uzunluk + self.genislik) # Dikdörtgen nesnesi oluşturma dikdortgen1 = Dikdortgen(5, 3) print(f"Alan: {dikdortgen1.alan_hesapla()}") # Alan: 15 print(f"Çevre: {dikdortgen1.cevre_hesapla()}") # Çevre: 16

Kalıtım (Inheritance)

Kalıtım, bir sınıfın başka bir sınıfın özelliklerini ve davranışlarını devralmasını sağlar. Bu, kod tekrarını azaltır ve hiyerarşik ilişkiler kurmanıza olanak tanır.

# Temel sınıf (Base class / Parent class) class Hayvan: def __init__(self, isim, yas): self.isim = isim self.yas = yas def ses_cikar(self): return "Bir ses çıkarıyor..." def bilgi_goster(self): return f"{self.isim}, {self.yas} yaşında" # Türetilmiş sınıf (Derived class / Child class) class Kopek(Hayvan): def __init__(self, isim, yas, cins): # Üst sınıfın yapıcı metodunu çağırma super().__init__(isim, yas) self.cins = cins # Metot ezme (Method overriding) def ses_cikar(self): return "Hav hav!" def bilgi_goster(self): return f"{super().bilgi_goster()}, {self.cins} cinsi" # Hayvan nesnesi oluşturma hayvan1 = Hayvan("Genel Hayvan", 5) print(hayvan1.bilgi_goster()) # Genel Hayvan, 5 yaşında print(hayvan1.ses_cikar()) # Bir ses çıkarıyor... # Köpek nesnesi oluşturma kopek1 = Kopek("Karabaş", 3, "Golden") print(kopek1.bilgi_goster()) # Karabaş, 3 yaşında, Golden cinsi print(kopek1.ses_cikar()) # Hav hav!

Kapsülleme (Encapsulation)

Kapsülleme, sınıfın içindeki verilerin gizlenmesini ve korunmasını sağlar. Python'da, değişkenlerin başına alt çizgi (_) veya çift alt çizgi (__) koyarak onları "gizli" hale getirebilirsiniz.

class BankaHesabi: def __init__(self, hesap_no, isim, bakiye): self.hesap_no = hesap_no self.isim = isim self.__bakiye = bakiye # Özel (private) değişken # Getter metodu def get_bakiye(self): return self.__bakiye # Setter metodu def set_bakiye(self, miktar): if miktar > 0: self.__bakiye = miktar else: print("Bakiye negatif olamaz!") def para_yatir(self, miktar): if miktar > 0: self.__bakiye += miktar print(f"{miktar} TL yatırıldı. Yeni bakiye: {self.__bakiye} TL") else: print("Geçersiz miktar!") def para_cek(self, miktar): if 0 < miktar <= self.__bakiye: self.__bakiye -= miktar print(f"{miktar} TL çekildi. Yeni bakiye: {self.__bakiye} TL") else: print("Geçersiz miktar veya yetersiz bakiye!") # Banka hesabı oluşturma hesap1 = BankaHesabi("123456", "Ali Veli", 1000) # Doğrudan erişim mümkün değil # print(hesap1.__bakiye) # AttributeError: 'BankaHesabi' object has no attribute '__bakiye' # Getter metoduyla erişim print(f"Bakiye: {hesap1.get_bakiye()} TL") # Bakiye: 1000 TL # İşlemler hesap1.para_yatir(500) # 500 TL yatırıldı. Yeni bakiye: 1500 TL hesap1.para_cek(300) # 300 TL çekildi. Yeni bakiye: 1200 TL

İsim Değiştirme (Name Mangling)

Python'da çift alt çizgi (__) ile başlayan değişkenler, sınıf dışından doğrudan erişimi engellemek için isim değiştirme (name mangling) işlemine tabi tutulur. Örneğin, __bakiye değişkeni aslında _BankaHesabi__bakiye olarak saklanır.

Çok Biçimlilik (Polymorphism)

Çok biçimlilik, farklı sınıfların aynı metod adını kullanarak farklı davranışlar sergilemesine olanak tanır. Bu, kodun daha esnek ve genişletilebilir olmasını sağlar.

class Sekil: def alan_hesapla(self): pass class Kare(Sekil): def __init__(self, kenar): self.kenar = kenar def alan_hesapla(self): return self.kenar ** 2 class Daire(Sekil): def __init__(self, yaricap): self.yaricap = yaricap def alan_hesapla(self): return 3.14 * self.yaricap ** 2 # Şekiller oluşturma kare = Kare(5) daire = Daire(3) # Çok biçimlilik sekiller = [kare, daire] for sekil in sekiller: print(f"Alan: {sekil.alan_hesapla()}") # Çıktı: # Alan: 25 # Alan: 28.26

Sınıf Değişkenleri ve Örnek Değişkenleri

Python'da iki tür değişken vardır:

  • Sınıf Değişkenleri (Class Variables): Tüm örnekler tarafından paylaşılan değişkenlerdir.
  • Örnek Değişkenleri (Instance Variables): Her örneğe özgü olan değişkenlerdir.
class Calisan: # Sınıf değişkeni sirket = "ABC Şirketi" calisan_sayisi = 0 def __init__(self, isim, maas): # Örnek değişkenleri self.isim = isim self.maas = maas Calisan.calisan_sayisi += 1 def bilgi_goster(self): return f"{self.isim}, {self.sirket}'de çalışıyor ve {self.maas} TL maaş alıyor." @classmethod def sirket_degistir(cls, yeni_sirket): cls.sirket = yeni_sirket # Çalışanlar oluşturma calisan1 = Calisan("Ahmet", 5000) calisan2 = Calisan("Mehmet", 6000) print(calisan1.bilgi_goster()) # Ahmet, ABC Şirketi'de çalışıyor ve 5000 TL maaş alıyor. print(calisan2.bilgi_goster()) # Mehmet, ABC Şirketi'de çalışıyor ve 6000 TL maaş alıyor. print(f"Toplam çalışan sayısı: {Calisan.calisan_sayisi}") # Toplam çalışan sayısı: 2 # Sınıf değişkenini değiştirme Calisan.sirket_degistir("XYZ Şirketi") print(calisan1.bilgi_goster()) # Ahmet, XYZ Şirketi'de çalışıyor ve 5000 TL maaş alıyor. print(calisan2.bilgi_goster()) # Mehmet, XYZ Şirketi'de çalışıyor ve 6000 TL maaş alıyor.

Özel Metotlar (Magic Methods / Dunder Methods)

Python sınıflarında, çift alt çizgi ile başlayıp biten özel metotlar vardır. Bu metotlar, sınıfın davranışını özelleştirmenize olanak tanır.

class Kitap: def __init__(self, baslik, yazar, sayfa_sayisi): self.baslik = baslik self.yazar = yazar self.sayfa_sayisi = sayfa_sayisi # Nesnenin string gösterimi def __str__(self): return f"{self.baslik} - {self.yazar}" # Nesnenin resmi string gösterimi def __repr__(self): return f"Kitap('{self.baslik}', '{self.yazar}', {self.sayfa_sayisi})" # İki nesnenin karşılaştırılması def __eq__(self, other): if not isinstance(other, Kitap): return False return (self.baslik == other.baslik and self.yazar == other.yazar) # Nesnelerin sıralanması def __lt__(self, other): if not isinstance(other, Kitap): return NotImplemented return self.sayfa_sayisi < other.sayfa_sayisi # İki nesnenin toplanması def __add__(self, other): if isinstance(other, Kitap): toplam_sayfa = self.sayfa_sayisi + other.sayfa_sayisi return f"{self.baslik} ve {other.baslik} kitaplarının toplam sayfa sayısı: {toplam_sayfa}" return NotImplemented # Kitap nesneleri oluşturma kitap1 = Kitap("Python Programlama", "Ahmet Yazar", 350) kitap2 = Kitap("Veri Bilimi", "Ayşe Yazar", 450) kitap3 = Kitap("Python Programlama", "Ahmet Yazar", 350) print(kitap1) # Python Programlama - Ahmet Yazar print(repr(kitap2)) # Kitap('Veri Bilimi', 'Ayşe Yazar', 450) print(kitap1 == kitap3) # True (aynı başlık ve yazar) print(kitap1 < kitap2) # True (kitap1'in sayfa sayısı daha az) print(kitap1 + kitap2) # Python Programlama ve Veri Bilimi kitaplarının toplam sayfa sayısı: 800

@property Dekoratörü

@property dekoratörü, bir metodu özellik (property) gibi kullanmanızı sağlar. Bu, değişkenlere daha kontrollü erişim sağlamak için kullanılır.

class Sicaklik: def __init__(self, celsius=0): self._celsius = celsius @property def celsius(self): return self._celsius @celsius.setter def celsius(self, deger): if deger < -273.15: raise ValueError("Sıcaklık mutlak sıfırdan düşük olamaz!") self._celsius = deger @property def fahrenheit(self): return (self._celsius * 9/5) + 32 @fahrenheit.setter def fahrenheit(self, deger): self.celsius = (deger - 32) * 5/9 # Sıcaklık nesnesi oluşturma sicaklik = Sicaklik(25) print(f"Celsius: {sicaklik.celsius}°C") # Celsius: 25°C print(f"Fahrenheit: {sicaklik.fahrenheit}°F") # Fahrenheit: 77.0°F # Fahrenheit olarak ayarlama sicaklik.fahrenheit = 86 print(f"Celsius: {sicaklik.celsius}°C") # Celsius: 30.0°C print(f"Fahrenheit: {sicaklik.fahrenheit}°F") # Fahrenheit: 86.0°F # Hata kontrolü try: sicaklik.celsius = -300 # ValueError: Sıcaklık mutlak sıfırdan düşük olamaz! except ValueError as e: print(e)

Soyut Sınıflar (Abstract Classes)

Soyut sınıflar, doğrudan örneklenemeyen ancak alt sınıflar için bir şablon görevi gören sınıflardır. Python'da, abc modülünü kullanarak soyut sınıflar ve metotlar tanımlayabilirsiniz.

from abc import ABC, abstractmethod class CizimAraci(ABC): @abstractmethod def ciz(self): pass @abstractmethod def boyut_ayarla(self, boyut): pass def rengini_goster(self): return "Siyah" # Zorunlu olmayan metot class Kalem(CizimAraci): def __init__(self, tip="Kurşun", boyut=1): self.tip = tip self.boyut = boyut def ciz(self): return f"{self.tip} kalem ile çiziliyor." def boyut_ayarla(self, boyut): self.boyut = boyut return f"Kalem boyutu {boyut} olarak ayarlandı." class Firca(CizimAraci): def __init__(self, tip="Yuvarlak", boyut=5): self.tip = tip self.boyut = boyut def ciz(self): return f"{self.tip} fırça ile çiziliyor." def boyut_ayarla(self, boyut): self.boyut = boyut return f"Fırça boyutu {boyut} olarak ayarlandı." def rengini_goster(self): return "Mavi" # Metodu ezme (override) # Soyut sınıfı örnekleyemezsiniz # arac = CizimAraci() # TypeError: Can't instantiate abstract class CizimAraci with abstract methods boyut_ayarla, ciz # Alt sınıfları örnekleme kalem = Kalem() firca = Firca() print(kalem.ciz()) # Kurşun kalem ile çiziliyor. print(firca.ciz()) # Yuvarlak fırça ile çiziliyor. print(kalem.rengini_goster()) # Siyah print(firca.rengini_goster()) # Mavi

Sonraki Adımlar

Python nesne yönelimli programlama konusunu tamamladıktan sonra öğrenmenize devam edebileceğiniz konular:

  • İleri Düzey OOP: Soyut sınıflar, arayüzler ve çoklu kalıtım konularını derinlemesine öğrenin.
  • Tasarım Desenleri: Singleton, Factory, Observer gibi yaygın tasarım desenlerini öğrenin.
  • Modül ve Paketler: Kendi modüllerinizi ve paketlerinizi nasıl oluşturacağınızı öğrenin.
  • Dosya İşlemleri ve Serileştirme: Nesneleri dosyalara nasıl kaydedeceğinizi ve yükleyeceğinizi öğrenin.
  • Hata Yakalama ve İstisnalar: OOP'de hata yönetimini ve özel istisnalar oluşturmayı öğrenin.

Başarı İpucu

Python'da nesne yönelimli programlamayı öğrenirken, gerçek dünya problemlerini modelleyerek küçük sınıflar ve uygulamalar yazın. Özellikle kalıtım, kapsülleme ve çok biçimlilik prensiplerine odaklanın ve bunları projelerinizde uygulamaya çalışın!
