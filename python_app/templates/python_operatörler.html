Python'da Operatörler

Bu dersimizde Python'da operatörler nedir, nasıl kullanılır ve farklı operatör türleri hakkında bilgi edineceksiniz.

Operatörler Nedir?

Operatörler, programlama dillerinde hesaplama yapmak veya değer karşılaştırmak için kullanılan sembollerdir. Bir operatör, değişkenler veya değerler (operand olarak adlandırılır) üzerinde belirli işlemler gerçekleştirir.

Python'da birçok farklı türde operatör bulunur ve bunlar sayesinde matematiksel işlemler, karşılaştırmalar, mantıksal değerlendirmeler ve daha fazlasını yapabilirsiniz.

# Operatör örneği a = 10 b = 3 # Aritmetik operatörler toplam = a + b # 13 fark = a - b # 7 carpim = a * b # 30 bolum = a / b # 3.3333... print("Toplam:", toplam) print("Fark:", fark) print("Çarpım:", carpim) print("Bölüm:", bolum)

Biliyor muydunuz?

Python'da bölme işlemi (/) her zaman ondalıklı sonuç üretir. Tam sayı bölümü için // operatörünü kullanabilirsiniz.

Python'da Operatör Türleri

Python'da operatörleri farklı kategorilere ayırabiliriz:

1. Aritmetik Operatörler

Aritmetik operatörler, matematiksel işlemler gerçekleştirmek için kullanılır.

  • + (Toplama): İki değeri toplar
  • - (Çıkarma): Bir değerden diğerini çıkarır
  • * (Çarpma): İki değeri çarpar
  • / (Bölme): Bir değeri diğerine böler (sonuç float türündedir)
  • // (Tam Sayı Bölme): Bölme işleminin tam sayı kısmını verir
  • % (Modül): Bölme işleminden kalanı verir
  • ** (Üs Alma): Bir sayının kuvvetini hesaplar
# Aritmetik operatörler a = 10 b = 3 print("a + b =", a + b) # 13 print("a - b =", a - b) # 7 print("a * b =", a * b) # 30 print("a / b =", a / b) # 3.3333... print("a // b =", a // b) # 3 (tam sayı bölümü) print("a % b =", a % b) # 1 (kalan) print("a ** b =", a ** b) # 1000 (10 üzeri 3)

Operatörlerin Metin İşlemlerde Kullanımı

Aritmetik operatörlerin bazıları metin (string) işlemleri için de kullanılabilir:

# String operasyonlarında + ve * kullanımı ad = "Python" soyad = "Programlama" # + operatörü string birleştirme için kullanılır tam_ad = ad + " " + soyad print(tam_ad) # "Python Programlama" # * operatörü string tekrarı için kullanılır tekrar = ad * 3 print(tekrar) # "PythonPythonPython"

2. Karşılaştırma Operatörleri

Karşılaştırma operatörleri, iki değeri karşılaştırmak için kullanılır ve sonuç olarak True (Doğru) veya False (Yanlış) döndürürler.

  • == (Eşittir): İki değer eşitse True, değilse False döndürür
  • != (Eşit Değildir): İki değer farklıysa True, aynıysa False döndürür
  • < (Küçüktür): Sol taraftaki değer sağdakinden küçükse True döndürür
  • > (Büyüktür): Sol taraftaki değer sağdakinden büyükse True döndürür
  • <= (Küçük veya Eşittir): Sol taraftaki değer sağdakinden küçük veya eşitse True döndürür
  • >= (Büyük veya Eşittir): Sol taraftaki değer sağdakinden büyük veya eşitse True döndürür
# Karşılaştırma operatörleri x = 5 y = 10 print("x == y:", x == y) # False print("x != y:", x != y) # True print("x < y:", x < y) # True print("x > y:", x > y) # False print("x <= y:", x <= y) # True print("x >= y:", x >= y) # False # String karşılaştırma ad1 = "Ahmet" ad2 = "Mehmet" print("ad1 == ad2:", ad1 == ad2) # False print("ad1 < ad2:", ad1 < ad2) # True (alfabetik sıralama)

3. Mantıksal Operatörler

Mantıksal operatörler, birden fazla koşulu birleştirmek veya tersine çevirmek için kullanılır.

  • and: Her iki koşul da True ise True döndürür
  • or: Koşullardan en az biri True ise True döndürür
  • not: Koşulun tersini döndürür (True ise False, False ise True)
# Mantıksal operatörler a = 5 b = 10 c = 15 # and operatörü print("a < b and b < c:", a < b and b < c) # True print("a < b and b > c:", a < b and b > c) # False # or operatörü print("a > b or b < c:", a > b or b < c) # True print("a > b or b > c:", a > b or b > c) # False # not operatörü print("not a > b:", not a > b) # True print("not a < b:", not a < b) # False

4. Atama Operatörleri

Atama operatörleri, değişkenlere değer atamak için kullanılır.

  • = (Atama): Sağdaki değeri soldaki değişkene atar
  • += (Topla ve Ata): Değişkene sağdaki değeri ekler ve sonucu atar
  • -= (Çıkar ve Ata): Değişkenden sağdaki değeri çıkarır ve sonucu atar
  • *= (Çarp ve Ata): Değişkeni sağdaki değerle çarpar ve sonucu atar
  • /= (Böl ve Ata): Değişkeni sağdaki değere böler ve sonucu atar
  • %= (Modül ve Ata): Değişkenin sağdaki değere bölümünden kalanı atar
  • **= (Üs al ve Ata): Değişkeni sağdaki değerin kuvvetine yükseltir ve sonucu atar
  • //= (Tam böl ve Ata): Değişkeni sağdaki değere tam böler ve sonucu atar
# Atama operatörleri x = 10 # x'e 10 değeri atanır # += operatörü x += 5 # x = x + 5 ile aynı print("x += 5:", x) # 15 # -= operatörü x -= 3 # x = x - 3 ile aynı print("x -= 3:", x) # 12 # *= operatörü x *= 2 # x = x * 2 ile aynı print("x *= 2:", x) # 24 # /= operatörü x /= 4 # x = x / 4 ile aynı print("x /= 4:", x) # 6.0 # String için += operatörü metin = "Python" metin += " Programlama" # metin = metin + " Programlama" ile aynı print(metin) # "Python Programlama"

5. Kimlik Operatörleri

Kimlik operatörleri, nesnelerin bellekte aynı nesneyi işaret edip etmediğini kontrol etmek için kullanılır.

  • is: İki değişken aynı nesneyi işaret ediyorsa True döndürür
  • is not: İki değişken farklı nesneleri işaret ediyorsa True döndürür
# Kimlik operatörleri a = [1, 2, 3] b = [1, 2, 3] c = a print("a is b:", a is b) # False (aynı değere sahip, ama farklı nesneler) print("a is c:", a is c) # True (aynı nesneyi işaret eder) print("a is not b:", a is not b) # True

6. Üyelik Operatörleri

Üyelik operatörleri, bir değerin bir dizide veya koleksiyonda olup olmadığını kontrol etmek için kullanılır.

  • in: Değer belirtilen dizide varsa True döndürür
  • not in: Değer belirtilen dizide yoksa True döndürür
# Üyelik operatörleri meyveler = ["elma", "armut", "muz", "portakal"] print("'elma' in meyveler:", "elma" in meyveler) # True print("'kiraz' in meyveler:", "kiraz" in meyveler) # False print("'kiraz' not in meyveler:", "kiraz" not in meyveler) # True # String için üyelik operatörü isim = "Python" print("'P' in isim:", "P" in isim) # True print("'z' not in isim:", "z" not in isim) # True

7. Bit (Bitwise) Operatörleri

Bit operatörleri, sayıların ikili (binary) gösterimleri üzerinde işlemler yapmak için kullanılır.

  • & (AND): Bit düzeyinde AND işlemi yapar
  • | (OR): Bit düzeyinde OR işlemi yapar
  • ^ (XOR): Bit düzeyinde XOR (exclusive OR) işlemi yapar
  • ~ (NOT): Bit düzeyinde NOT işlemi yapar (bitleri tersine çevirir)
  • << (Sol Kaydırma): Bitleri sola kaydırır
  • >> (Sağ Kaydırma): Bitleri sağa kaydırır
# Bit operatörleri a = 10 # İkili: 1010 b = 4 # İkili: 0100 print("a & b:", a & b) # 0 (İkili: 0000) print("a | b:", a | b) # 14 (İkili: 1110) print("a ^ b:", a ^ b) # 14 (İkili: 1110) print("~a:", ~a) # -11 (İkili: ...11110101) print("a << 1:", a << 1) # 20 (İkili: 10100) print("a >> 1:", a >> 1) # 5 (İkili: 0101)

Operatörlerin Önceliği

Python'da operatörler belirli bir öncelik sırasına göre değerlendirilir. Bu, bir ifadede birden fazla operatör olduğunda hangi işlemin önce yapılacağını belirler.

Operatör önceliği en yüksekten en düşüğe doğru şu şekildedir:

  1. () (Parantezler): İfadeleri gruplamak için kullanılır
  2. ** (Üs alma)
  3. +x, -x, ~x (Unary operatörler): Tekli artı, eksi ve bit NOT
  4. *, /, //, %: Çarpma, bölme, tam bölme, modül
  5. +, -: Toplama, çıkarma
  6. <<, >>: Bit kaydırma
  7. &: Bit AND
  8. ^: Bit XOR
  9. |: Bit OR
  10. ==, !=, <, <=, >, >=, is, is not, in, not in: Karşılaştırma, kimlik ve üyelik operatörleri
  11. not: Mantıksal NOT
  12. and: Mantıksal AND
  13. or: Mantıksal OR
# Operatör önceliği örnekleri sonuc1 = 10 + 3 * 2 # 16 (çarpma önce yapılır) sonuc2 = (10 + 3) * 2 # 26 (parantez içi önce hesaplanır) sonuc3 = 20 / 4 + 6 * 2 # 17.0 (önce bölme ve çarpma, sonra toplama) sonuc4 = 5 + 3 > 2 * 4 # False (önce aritmetik, sonra karşılaştırma) sonuc5 = not (5 > 3 and 2 < 1) # True (önce parantez içi, sonra not) print("10 + 3 * 2 =", sonuc1) print("(10 + 3) * 2 =", sonuc2) print("20 / 4 + 6 * 2 =", sonuc3) print("5 + 3 > 2 * 4 =", sonuc4) print("not (5 > 3 and 2 < 1) =", sonuc5)

İpucu

Karmaşık ifadelerde, operatör önceliğinden şüphe duyduğunuzda parantez kullanın. Bu hem kodunuzu daha okunaklı hale getirir hem de öncelik konusunda hata yapma riskini azaltır.
