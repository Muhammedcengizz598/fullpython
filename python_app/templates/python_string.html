Python'da String İşlemleri

Bu dersimizde Python'da string (metin) işlemleri, metotları ve kullanım örneklerini öğreneceksiniz.

String Nedir?

String, Python'da metinsel verileri temsil eden veri tipidir. Stringler, tek tırnak (' '), çift tırnak (" ") veya üç tırnak (''' ''' veya """ """) içinde tanımlanabilir.

Python'da stringler değiştirilemez (immutable) veri tipleridir. Yani bir string oluşturulduktan sonra, içeriği değiştirilemez.

# String tanımlama ad = 'Ahmet' soyad = "Yılmaz" adres = '''İstanbul Kadıköy Türkiye''' # Stringleri ekrana yazdırma print(ad) print(soyad) print(adres)

Biliyor muydunuz?

Python'da üç tırnak ile tanımlanan stringler, birden fazla satır içerebilir ve satır sonlarını korur.

String İndeksleme ve Dilimleme

Python'da stringler, karakter dizileri olarak ele alınır ve her karaktere indeks numarası ile erişilebilir:

# String indeksleme metin = "Python" print(metin[0]) # 'P' print(metin[1]) # 'y' print(metin[-1]) # 'n' (sondan başlayarak) print(metin[-2]) # 'o' # String dilimleme (slicing) print(metin[0:2]) # 'Py' (0'dan 2'ye kadar, 2 dahil değil) print(metin[2:4]) # 'th' print(metin[:3]) # 'Pyt' (baştan 3'e kadar) print(metin[3:]) # 'hon' (3'ten sona kadar) print(metin[-3:]) # 'hon' (sondan 3 karakter) print(metin[::2]) # 'Pto' (2'şer adımla)

String Birleştirme ve Çoğaltma

Stringleri birleştirmek için + operatörü, çoğaltmak için * operatörü kullanılır:

# String birleştirme ad = "Ahmet" soyad = "Yılmaz" tam_ad = ad + " " + soyad print(tam_ad) # 'Ahmet Yılmaz' # String çoğaltma tekrar = "Python " * 3 print(tekrar) # 'Python Python Python '

String Formatlama

Python'da string formatlama için üç farklı yöntem vardır:

1. % Operatörü ile Formatlama (Eski yöntem)

ad = "Ahmet" yas = 25 meslek = "mühendis" # % operatörü ile formatlama metin = "Merhaba, benim adım %s, %d yaşındayım ve %s olarak çalışıyorum." % (ad, yas, meslek) print(metin) # 'Merhaba, benim adım Ahmet, 25 yaşındayım ve mühendis olarak çalışıyorum.'

2. format() Metodu ile Formatlama

ad = "Ahmet" yas = 25 meslek = "mühendis" # format() metodu ile formatlama metin1 = "Merhaba, benim adım {}, {} yaşındayım ve {} olarak çalışıyorum.".format(ad, yas, meslek) print(metin1) # İndeks numarası ile formatlama metin2 = "Merhaba, benim adım {0}, {1} yaşındayım ve {2} olarak çalışıyorum.".format(ad, yas, meslek) print(metin2) # İsimlendirilmiş parametreler ile formatlama metin3 = "Merhaba, benim adım {a}, {y} yaşındayım ve {m} olarak çalışıyorum.".format(a=ad, y=yas, m=meslek) print(metin3)

3. f-string (Python 3.6+)

ad = "Ahmet" yas = 25 meslek = "mühendis" # f-string ile formatlama (en modern yöntem) metin = f"Merhaba, benim adım {ad}, {yas} yaşındayım ve {meslek} olarak çalışıyorum." print(metin) # f-string içinde işlemler yapabilirsiniz print(f"5 yıl sonra {yas + 5} yaşında olacağım.") print(f"Adımın uzunluğu: {len(ad)} karakter")

Modern String Formatlama

f-string, Python 3.6 ve üzeri sürümlerde kullanılabilen en modern ve okunması kolay string formatlama yöntemidir. Değişkenleri doğrudan süslü parantezler içinde kullanabilir ve karmaşık ifadeler yazabilirsiniz.

Yaygın String Metotları

Python'da stringleri işlemek için birçok yerleşik metot bulunur:

1. Büyük/Küçük Harf Dönüşümleri

metin = "Python Programlama" print(metin.upper()) # 'PYTHON PROGRAMLAMA' print(metin.lower()) # 'python programlama' print(metin.title()) # 'Python Programlama' (Her Kelimenin İlk Harfi Büyük) print(metin.capitalize()) # 'Python programlama' (Sadece İlk Harf Büyük) print(metin.swapcase()) # 'pYTHON pROGRAMLAMA' (Büyük/Küçük Harfleri Değiştirir)

2. Arama ve Değiştirme

metin = "Python Programlama Dili" # Arama metotları print(metin.find("Pro")) # 7 (ilk bulduğu indeksi döndürür) print(metin.find("Java")) # -1 (bulamazsa -1 döndürür) print(metin.index("Pro")) # 7 (find gibidir ama bulamazsa hata verir) print("Pro" in metin) # True (metin içinde varsa True döndürür) # Değiştirme yeni_metin = metin.replace("Python", "Java") print(yeni_metin) # 'Java Programlama Dili'

3. Bölme ve Birleştirme

# Bölme (split) metin = "Python,Java,C++,JavaScript" diller = metin.split(",") print(diller) # ['Python', 'Java', 'C++', 'JavaScript'] # Birleştirme (join) dil_listesi = ["Python", "Java", "C++", "JavaScript"] birlesik = ", ".join(dil_listesi) print(birlesik) # 'Python, Java, C++, JavaScript'

4. Boşluk İşlemleri

metin = " Python Programlama " print(metin.strip()) # 'Python Programlama' (baştaki ve sondaki boşlukları kaldırır) print(metin.lstrip()) # 'Python Programlama ' (baştaki boşlukları kaldırır) print(metin.rstrip()) # ' Python Programlama' (sondaki boşlukları kaldırır)

5. Kontrol Metotları

print("abc123".isalnum()) # True (alfanümerik mi?) print("123".isdigit()) # True (rakam mı?) print("abc".isalpha()) # True (harf mi?) print("ABC".isupper()) # True (büyük harf mi?) print("abc".islower()) # True (küçük harf mi?) print(" ".isspace()) # True (boşluk mu?) print("Python".startswith("P")) # True (P ile mi başlıyor?) print("Python".endswith("n")) # True (n ile mi bitiyor?)

String Escape Karakterleri

Python'da özel karakterleri string içinde kullanmak için escape karakterleri kullanılır:

print("Python'da string kullanımı") # Python'da string kullanımı print('Python\'da string kullanımı') # Python'da string kullanımı print("Satır 1\nSatır 2") # Satır 1 # Satır 2 print("Tab\tkarakter") # Tab karakter print("C:\\Program Files\\Python") # C:\Program Files\Python print(r"C:\Program Files\Python") # C:\Program Files\Python (raw string)

İpucu

Python'da r"..." şeklinde tanımlanan "raw string" (ham string) ifadelerinde escape karakterleri işlenmez. Bu özellikle dosya yollarını veya regex ifadelerini tanımlarken kullanışlıdır.

Python String İşlemleri

Python'da string (metin) işlemleri, metin verilerini manipüle etmek, analiz etmek ve dönüştürmek için kullanılan temel bir konudur. Bu derste, Python'da string veri tipi ve bu veri tipi üzerinde gerçekleştirebileceğiniz çeşitli işlemleri öğreneceksiniz.

Bu Derste Öğrenecekleriniz

  • String oluşturma ve tanımlama
  • String indeksleme ve dilimleme
  • String metotları ve özellikleri
  • String formatlama teknikleri
  • String manipülasyon işlemleri

String Oluşturma ve Tanımlama

Python'da string'ler tek tırnak, çift tırnak veya üç tırnak içinde tanımlanabilir: 

# Tek tırnak kullanımı
tek_tirnak = 'Merhaba Python'

# Çift tırnak kullanımı
cift_tirnak = "Merhaba Python"

# Üç tırnak (çoklu satır) kullanımı
coklu_satir = """Bu bir
çok satırlı
string örneğidir."""

print(tek_tirnak)
print(cift_tirnak)
print(coklu_satir)

String'ler değiştirilemez (immutable) veri tipleridir. Bir kez oluşturulduktan sonra içeriği değiştirilemez, ancak yeni string'ler oluşturulabilir.

String İndeksleme ve Dilimleme

Python'da string'lerin her karakterine indeks numarası ile erişebilirsiniz. İndeksleme 0'dan başlar. 

                metin = "Python Programlama"

# İndeksleme
print(metin[0])    # P
print(metin[7])    # P
print(metin[-1])   # a (sondan 1. karakter)

# Dilimleme (slicing)
print(metin[0:6])   # Python
print(metin[7:])    # Programlama
print(metin[:6])    # Python
print(metin[-11:])  # Programlama
print(metin[::2])   # Pto rgalm (2 adım atlayarak)

İpucu

String dilimlerken, metin[başlangıç:bitiş:adım] formatını kullanırsınız. Başlangıç dahil, bitiş hariçtir.

String Metotları

Python'da string'ler üzerinde işlem yapmak için birçok kullanışlı metot bulunmaktadır: 

metin = "   Python Programlama   "

# Temel metotlar
print(metin.upper())          # PYTHON PROGRAMLAMA
print(metin.lower())          # python programlama
print(metin.strip())          # "Python Programlama" (baştaki ve sondaki boşlukları siler)
print(metin.replace("P", "J")) # "   Jython Jrogramlama   "

# Arama metotları
print(metin.find("Pro"))      # 10 (ilk bulduğu konumu döndürür)
print(metin.count("a"))       # 3 (kaç tane 'a' olduğunu sayar)
print("Pro" in metin)         # True (içeriyor mu kontrolü)

# Bölme ve birleştirme
kelimeler = metin.strip().split(" ")  # ["Python", "Programlama"]
print("-".join(kelimeler))    # "Python-Programlama"

Dikkat

String metotları orijinal string'i değiştirmez, yeni bir string oluşturur. Çünkü string'ler değiştirilemez (immutable) veri tipleridir.

String Formatlama

Python'da string formatlama için üç farklı yöntem bulunmaktadır: 

isim = "Ahmet"
yas = 25
boy = 1.78

# 1. format() metodu
print("Merhaba, benim adım {}, {} yaşındayım ve boyum {} metredir.".format(isim, yas, boy))

# 2. f-string (Python 3.6+)
print(f"Merhaba, benim adım {isim}, {yas} yaşındayım ve boyum {boy} metredir.")

# 3. % operatörü (eski yöntem)
print("Merhaba, benim adım %s, %d yaşındayım ve boyum %.2f metredir." % (isim, yas, boy))

# Sayısal formatlama
pi = 3.14159
print(f"Pi sayısı: {pi:.2f}")  # Pi sayısı: 3.14
print(f"Pi sayısı: {pi:10.2f}")  # Pi sayısı:       3.14 (10 karakter genişliğinde)

F-string'ler (Python 3.6+) en modern ve okunabilir formatlama yöntemidir. String içerisinde doğrudan değişken ve ifadeler kullanabilirsiniz.

String İşlemleri

String'ler üzerinde çeşitli işlemler gerçekleştirebilirsiniz: 

# Birleştirme (concatenation)
ad = "Python"
soyad = "Programlama"
tam_ad = ad + " " + soyad  # "Python Programlama"

# Tekrarlama
tekrar = "Python" * 3  # "PythonPythonPython"

# Karşılaştırma
print("abc" < "def")  # True (alfabetik sıralama)
print("Python" == "python")  # False (büyük/küçük harf duyarlı)
print("Python".lower() == "python")  # True

# Uzunluk
print(len("Python"))  # 6

# Karakter değişimi (doğrudan değil, yeni string oluşturarak)
s = "Python"
s_yeni = s[:2] + "x" + s[3:]  # "Pyxhon"

Başarı İpucu
