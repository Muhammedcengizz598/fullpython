Python'da Sözlükler (Dictionaries)

Bu dersimizde Python'da sözlükler (dictionaries) nedir, nasıl tanımlanır ve kullanılır konusunda detaylı bilgi edineceksiniz.

Sözlükler Nedir?

Sözlükler (dictionaries), anahtar-değer (key-value) çiftlerini saklamak için kullanılan koleksiyon veri tipidir. Günlük hayattaki sözlüklere benzer şekilde, kelimeler ve anlamları gibi ilişkili verileri bir arada tutarlar.

Python'da sözlükler süslü parantezler {} içinde tanımlanır ve her bir anahtar-değer çifti iki nokta üst üste (:) ile ayrılır.

# Basit bir sözlük tanımlama kisi = { "ad": "Ahmet", "yas": 25, "meslek": "Mühendis", "ogrenci_mi": False } # Sözlük verilerine erişim print("Ad:", kisi["ad"]) print("Yaş:", kisi["yas"]) print("Meslek:", kisi["meslek"]) print("Öğrenci mi?", kisi["ogrenci_mi"])

Biliyor muydunuz?

Python'da sözlük anahtarları değiştirilemez (immutable) veri tiplerinden olmalıdır. Bu nedenle string, integer, float, tuple gibi değiştirilemez veri tipleri anahtar olarak kullanılabilirken, liste (list) veya sözlük (dictionary) gibi değiştirilebilir veri tipleri anahtar olarak kullanılamaz.

Sözlük Oluşturma Yöntemleri

Python'da sözlük oluşturmak için çeşitli yöntemler vardır:

# 1. Süslü parantezler kullanarak sozluk1 = {"anahtar1": "değer1", "anahtar2": "değer2"} # 2. dict() fonksiyonu kullanarak sozluk2 = dict(anahtar1="değer1", anahtar2="değer2") # 3. Liste içinde tuple'lardan oluşturma ciftler = [("anahtar1", "değer1"), ("anahtar2", "değer2")] sozluk3 = dict(ciftler) # 4. Sözlük oluşturucu (dictionary comprehension) kullanarak sozluk4 = {x: x**2 for x in range(5)} # {0:0, 1:1, 2:4, 3:9, 4:16} # 5. Boş bir sözlük oluşturma bos_sozluk = {}

Sözlük Elemanlarına Erişim

Sözlük elemanlarına erişmek için iki temel yöntem vardır:

ogretmen = { "ad": "Zeynep", "soyad": "Yılmaz", "ders": "Matematik", "yas": 32 } # 1. Köşeli parantez kullanarak print(ogretmen["ad"]) # Zeynep print(ogretmen["ders"]) # Matematik # Olmayan bir anahtara erişmeye çalışırsanız KeyError hatası alırsınız # print(ogretmen["okul"]) # KeyError # 2. get() metodu kullanarak print(ogretmen.get("ad")) # Zeynep print(ogretmen.get("okul")) # None (hata vermez) print(ogretmen.get("okul", "Bilgi yok")) # "Bilgi yok" (varsayılan değer tanımlanabilir)

İpucu

get() metodu, anahtarın sözlükte olmaması durumunda hata vermek yerine None değeri döndürür. İkinci bir parametre ekleyerek varsayılan bir değer de belirleyebilirsiniz.

Sözlükleri Güncelleme

Python'da sözlükleri güncellemek için çeşitli yöntemler kullanılır:

araba = { "marka": "Toyota", "model": "Corolla", "yil": 2020 } # Var olan bir değeri güncelleme araba["yil"] = 2021 print(araba) # {'marka': 'Toyota', 'model': 'Corolla', 'yil': 2021} # Yeni bir anahtar-değer çifti ekleme araba["renk"] = "Beyaz" print(araba) # {'marka': 'Toyota', 'model': 'Corolla', 'yil': 2021, 'renk': 'Beyaz'} # update() metodu ile birden fazla değeri aynı anda güncelleme araba.update({"yil": 2022, "km": 15000, "renk": "Siyah"}) print(araba) # {'marka': 'Toyota', 'model': 'Corolla', 'yil': 2022, 'renk': 'Siyah', 'km': 15000}

Sözlüklerden Eleman Silme

Sözlüklerden eleman silmek için birkaç farklı yöntem kullanabilirsiniz:

telefon = { "marka": "Apple", "model": "iPhone 13", "yil": 2021, "hafiza": "128GB", "renk": "Mavi" } # 1. pop() metodu - belirtilen anahtarı siler ve değerini döndürür hafiza = telefon.pop("hafiza") print("Silinen değer:", hafiza) # Silinen değer: 128GB print(telefon) # {'marka': 'Apple', 'model': 'iPhone 13', 'yil': 2021, 'renk': 'Mavi'} # 2. del anahtar kelimesi ile silme del telefon["yil"] print(telefon) # {'marka': 'Apple', 'model': 'iPhone 13', 'renk': 'Mavi'} # 3. popitem() - son eklenen anahtar-değer çiftini siler ve döndürür (Python 3.7+) son_cift = telefon.popitem() print("Son çift:", son_cift) # Son çift: ('renk', 'Mavi') print(telefon) # {'marka': 'Apple', 'model': 'iPhone 13'} # 4. clear() - tüm sözlüğü temizler telefon.clear() print(telefon) # {}

Sözlükleri Döngülerle Kullanma

Sözlükleri döngüler içinde kullanmak için çeşitli yöntemler vardır:

ogrenci = { "ad": "Ayşe", "soyad": "Kaya", "no": 101, "dersler": ["Matematik", "Fizik", "Kimya"], "not_ortalama": 85.6 } # 1. Anahtarları döngüde kullanma print("Anahtarlar:") for anahtar in ogrenci: print(anahtar) # 2. keys() metodu ile anahtarları döngüde kullanma print("\nkeys() ile anahtarlar:") for anahtar in ogrenci.keys(): print(anahtar) # 3. values() metodu ile değerleri döngüde kullanma print("\nDeğerler:") for deger in ogrenci.values(): print(deger) # 4. items() metodu ile anahtar-değer çiftlerini döngüde kullanma print("\nAnahtar-Değer çiftleri:") for anahtar, deger in ogrenci.items(): print(f"{anahtar}: {deger}")

Sözlüklerde Anahtar Kontrolü

Python'da bir anahtarın sözlükte olup olmadığını kontrol etmek için in operatörünü kullanabilirsiniz:

bilgisayar = { "islemci": "Intel i7", "ram": "16GB", "disk": "512GB SSD" } # Anahtarın varlığını kontrol etme if "islemci" in bilgisayar: print("İşlemci bilgisi mevcut:", bilgisayar["islemci"]) if "ekran_karti" not in bilgisayar: print("Ekran kartı bilgisi mevcut değil") # Yeni bir değer ekleyebiliriz bilgisayar["ekran_karti"] = "NVIDIA RTX 3060" print(bilgisayar)

İç İçe Sözlükler (Nested Dictionaries)

Sözlükler içinde başka sözlükler de saklayabilirsiniz, bu tür yapılara iç içe sözlükler (nested dictionaries) denir:

okul = { "sinif_1A": { "ogretmen": "Ahmet Yıldız", "ogrenciler": ["Ali", "Veli", "Ayşe"], "ders_saati": 30 }, "sinif_1B": { "ogretmen": "Zeynep Kara", "ogrenciler": ["Mehmet", "Fatma", "Can"], "ders_saati": 28 }, "mudur": "Hasan Şahin" } # İç içe sözlük verilerine erişim print("1A sınıfı öğretmeni:", okul["sinif_1A"]["ogretmen"]) print("1B sınıfı öğrencileri:", okul["sinif_1B"]["ogrenciler"]) print("Okul müdürü:", okul["mudur"]) # İç içe sözlüklerde değişiklik yapma okul["sinif_1A"]["ogrenciler"].append("Deniz") print("1A sınıfı güncel öğrenci listesi:", okul["sinif_1A"]["ogrenciler"]) # Yeni bir iç sözlük ekleme okul["sinif_1C"] = { "ogretmen": "Melis Aydın", "ogrenciler": ["Ece", "Emre"], "ders_saati": 25 } print("Okuldaki sınıflar:", list(okul.keys()))

Sözlük Metotları

Python'da sözlüklerle çalışırken kullanabileceğiniz önemli metotlar:

kitap = { "baslik": "Python Programlama", "yazar": "Ahmet Öz", "yil": 2022, "sayfa": 450 } # 1. keys() - anahtarların bir görünümünü (view) döndürür print("Anahtarlar:", kitap.keys()) # 2. values() - değerlerin bir görünümünü (view) döndürür print("Değerler:", kitap.values()) # 3. items() - anahtar-değer çiftlerinin bir görünümünü (view) döndürür print("Çiftler:", kitap.items()) # 4. get() - anahtara karşılık gelen değeri döndürür print("Başlık:", kitap.get("baslik")) print("Fiyat:", kitap.get("fiyat", "Fiyat bilgisi yok")) # 5. update() - sözlüğü başka bir sözlükle günceller kitap.update({"fiyat": 75, "dil": "Türkçe"}) print("Güncellenmiş kitap:", kitap) # 6. setdefault() - anahtar yoksa ekler ve değerini döndürür, varsa değerini değiştirmez kitap.setdefault("yayin_evi", "ABC Yayınları") # Anahtar yoktu, ekledi kitap.setdefault("yil", 2023) # Anahtar vardı, değiştirmedi print("setdefault sonrası:", kitap) # 7. copy() - sözlüğün bir kopyasını oluşturur kitap_kopya = kitap.copy() kitap_kopya["baslik"] = "Python İleri Programlama" print("Orijinal:", kitap["baslik"]) print("Kopya:", kitap_kopya["baslik"])

Unutmayın!

dict_keys, dict_values ve dict_items nesneleri "görünüm" (view) nesneleridir ve orijinal sözlükteki değişiklikleri otomatik olarak yansıtırlar. Ayrıca bunlar üzerinde doğrudan değişiklik yapamazsınız, ancak list() fonksiyonu ile listeye dönüştürebilirsiniz.

Sözlük Oluşturucular (Dictionary Comprehensions)

Sözlük oluşturucular (dictionary comprehensions), tek satırlık ifadeler kullanarak hızlı bir şekilde sözlük oluşturmanıza olanak sağlar:

# 1. Basit bir sözlük oluşturucu kareler = {x: x**2 for x in range(1, 6)} print("Sayıların kareleri:", kareler) # {1: 1, 2: 4, 3: 9, 4: 16, 5: 25} # 2. Koşullu sözlük oluşturucu cift_kareler = {x: x**2 for x in range(1, 11) if x % 2 == 0} print("Çift sayıların kareleri:", cift_kareler) # {2: 4, 4: 16, 6: 36, 8: 64, 10: 100} # 3. Mevcut bir sözlükten yeni sözlük oluşturma fiyatlar = {"elma": 5, "armut": 7, "muz": 8, "kiraz": 12} indirimli_fiyatlar = {meyve: fiyat * 0.9 for meyve, fiyat in fiyatlar.items()} print("İndirimli fiyatlar:", indirimli_fiyatlar) # 4. Koşullu filtreleme uygun_fiyatlar = {meyve: fiyat for meyve, fiyat in fiyatlar.items() if fiyat < 10} print("Uygun fiyatlı meyveler:", uygun_fiyatlar)

Sözlüklerde Sıralama İşlemleri

Python 3.7'den itibaren sözlükler ekleme sırasını korurlar, ancak sözlükleri farklı kriterlere göre sıralamak için built-in fonksiyonlar kullanabilirsiniz:

ogrenciler = { "Ali": 85, "Veli": 92, "Ayşe": 78, "Fatma": 95, "Mehmet": 63 } # 1. Anahtarlara göre sıralama (alfabetik) print("İsme göre sıralama:") for isim in sorted(ogrenciler.keys()): print(f"{isim}: {ogrenciler[isim]}") # 2. Değerlere göre sıralama (notlara göre) print("\nNota göre sıralama (küçükten büyüğe):") for isim, not_degeri in sorted(ogrenciler.items(), key=lambda x: x[1]): print(f"{isim}: {not_degeri}") # 3. Değerlere göre tersten sıralama (notlara göre, büyükten küçüğe) print("\nNota göre sıralama (büyükten küçüğe):") for isim, not_degeri in sorted(ogrenciler.items(), key=lambda x: x[1], reverse=True): print(f"{isim}: {not_degeri}") # 4. Anahtarların uzunluğuna göre sıralama print("\nİsim uzunluğuna göre sıralama:") for isim, not_degeri in sorted(ogrenciler.items(), key=lambda x: len(x[0])): print(f"{isim}: {not_degeri}")

Sözlük Kullanımı İçin Öneriler

Sözlükleri etkili bir şekilde kullanmak için bazı öneriler:

  • Anlamlı anahtarlar kullanın: Anahtarlar, verinin ne olduğunu açıkça belirtmelidir.
  • get() metodunu tercih edin: Var olmayan bir anahtara erişirken, KeyError hatası almamak için get() metodunu kullanın.
  • Varsayılan sözlükleri düşünün: Çok sayıda iç içe yapı için collections modülündeki defaultdict kullanmayı düşünün.
  • Anahtar kontrolünü unutmayın: Bir anahtarın sözlükte olup olmadığını kontrol etmek için in operatörünü kullanın.
  • Sözlük görünümlerini kullanın: keys(), values() ve items() metotları dinamik görünümler sağlar ve bellek kullanımını optimize eder.
# Varsayılan sözlük (defaultdict) örneği from collections import defaultdict # Bir sayının kaç kez geçtiğini sayan program sayilar = [1, 2, 3, 4, 2, 3, 1, 4, 4, 5] # Normal sözlük ile frekans1 = {} for sayi in sayilar: if sayi in frekans1: frekans1[sayi] += 1 else: frekans1[sayi] = 1 # defaultdict ile (daha kısa ve okunabilir) frekans2 = defaultdict(int) # Varsayılan değer 0 olacak for sayi in sayilar: frekans2[sayi] += 1 print("Normal sözlük:", dict(frekans1)) print("defaultdict:", dict(frekans2))

İleri Seviye: Counter Kullanımı

Elemanların sayısını saymak için collections modülünde bulunan Counter sınıfını da kullanabilirsiniz:

from collections import Counter sayilar = [1, 2, 3, 4, 2, 3, 1, 4, 4, 5] frekans = Counter(sayilar) print(frekans) # Counter({4: 3, 1: 2, 2: 2, 3: 2, 5: 1}) # En sık geçen elemanları bulma print("En sık geçen 2 eleman:", frekans.most_common(2)) # [(4, 3), (1, 2)]
