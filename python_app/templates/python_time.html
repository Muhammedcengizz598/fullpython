Python'da Tarih ve Zaman İşlemleri

Bu dersimizde Python'da tarih ve zaman işlemleri, datetime modülü ve farklı tarih formatlama teknikleri hakkında bilgi edineceksiniz.

Python'da Tarih ve Zaman Modülleri

Python'da tarih ve zaman işlemleri için birkaç farklı modül bulunmaktadır. En yaygın kullanılanları datetime, time ve calendar modülleridir. Bu modüller sayesinde tarih oluşturma, zaman hesaplama, formatlama gibi işlemleri kolayca yapabilirsiniz.

Bu derste ağırlıklı olarak datetime modülünü inceleyeceğiz, çünkü bu modül tarih ve zaman işlemleri için en kapsamlı araçları sağlar.

# Gerekli modülleri içe aktarma import datetime import time import calendar # Basit bir örnek print("Şu anki tarih ve saat:", datetime.datetime.now())

Biliyor muydunuz?

Python'da datetime modülü, Unix zaman damgası 1 Ocak 1970'den önceki ve sonraki tarihleri işleyebilir. Böylece tarihsel olayları veya gelecekteki tarihleri programınızda kullanabilirsiniz.

datetime Modülü ve Sınıfları

datetime modülü içinde farklı amaçlar için kullanabileceğiniz birkaç sınıf bulunur:

  • datetime.date: Yıl, ay ve gün bilgilerini içerir.
  • datetime.time: Saat, dakika, saniye ve mikrosaniye bilgilerini içerir.
  • datetime.datetime: Hem tarih hem de zaman bilgilerini içerir.
  • datetime.timedelta: İki tarih veya zaman arasındaki farkı temsil eder.
  • datetime.tzinfo: Saat dilimi bilgisi için soyut taban sınıftır.
# datetime.date sınıfı örneği from datetime import date bugun = date.today() print("Bugünün tarihi:", bugun) print("Yıl:", bugun.year) print("Ay:", bugun.month) print("Gün:", bugun.day) # Belirli bir tarih oluşturma dogum_gunu = date(1990, 5, 15) print("Doğum günü:", dogum_gunu)

Şu Anki Tarih ve Zaman

Python'da şu anki tarih ve zamanı almanın birkaç yolu vardır:

from datetime import datetime # Şu anki tarih ve zaman simdi = datetime.now() print("Şu anki tarih ve saat:", simdi) # UTC zamanı utc_simdi = datetime.utcnow() print("UTC zamanı:", utc_simdi) # time modülü ile Unix zaman damgası almak import time zaman_damgasi = time.time() print("Unix zaman damgası:", zaman_damgasi)

1. Tarih ve Zaman Oluşturma

  • datetime.date(yil, ay, gun): Belirli bir tarihi oluşturur.
  • datetime.time(saat, dakika, saniye, mikrosaniye): Belirli bir zamanı oluşturur.
  • datetime.datetime(yil, ay, gun, saat, dakika, saniye, mikrosaniye): Belirli bir tarih ve zamanı oluşturur.
from datetime import date, time, datetime # Belirli bir tarih oluşturma tarih = date(2023, 12, 25) print("Oluşturulan tarih:", tarih) # Belirli bir zaman oluşturma zaman = time(13, 30, 45) print("Oluşturulan zaman:", zaman) # Tarih ve zamanı birleştirme tarih_zaman = datetime(2023, 12, 25, 13, 30, 45) print("Oluşturulan tarih ve zaman:", tarih_zaman)

2. Tarihler Arası Fark Hesaplama

  • datetime.timedelta: İki tarih arasındaki farkı hesaplamak veya bir tarihe belirli bir süre eklemek/çıkarmak için kullanılır.
from datetime import datetime, timedelta # Şu anki zaman simdi = datetime.now() print("Şu an:", simdi) # 5 gün sonrası bes_gun_sonra = simdi + timedelta(days=5) print("5 gün sonra:", bes_gun_sonra) # 3 saat öncesi uc_saat_once = simdi - timedelta(hours=3) print("3 saat önce:", uc_saat_once) # İki tarih arasındaki farkı hesaplama dogum_tarihi = datetime(1990, 5, 15) yas_timedelta = simdi - dogum_tarihi print("Gün olarak yaş:", yas_timedelta.days) print("Saniye olarak yaş:", yas_timedelta.total_seconds())

3. Tarih ve Zaman Formatlama

  • strftime(): Tarih ve zaman nesnelerini formatlı bir metne dönüştürür.
  • strptime(): Formatlı bir metni tarih ve zaman nesnesine dönüştürür.
from datetime import datetime # Şu anki zaman simdi = datetime.now() # strftime ile formatlama tarih_str = simdi.strftime("%d/%m/%Y") print("Formatlanmış tarih (gün/ay/yıl):", tarih_str) zaman_str = simdi.strftime("%H:%M:%S") print("Formatlanmış zaman (saat:dakika:saniye):", zaman_str) tam_str = simdi.strftime("%d %B %Y, %A %H:%M") print("Uzun format:", tam_str) # strptime ile metin tarihini datetime nesnesine dönüştürme tarih_metin = "15-05-1990" tarih_nesnesi = datetime.strptime(tarih_metin, "%d-%m-%Y") print("Dönüştürülen tarih nesnesi:", tarih_nesnesi)

Formatlama Kod Karakterleri

strftime() ve strptime() metodlarında kullanabileceğiniz bazı yaygın format kodları:

# %d - Gün (01-31) # %m - Ay (01-12) # %Y - 4 basamaklı yıl (örn. 2023) # %y - 2 basamaklı yıl (örn. 23) # %H - Saat, 24 saat formatında (00-23) # %I - Saat, 12 saat formatında (01-12) # %M - Dakika (00-59) # %S - Saniye (00-59) # %A - Haftanın günü, tam ad (örn. Pazartesi) # %a - Haftanın günü, kısa ad (örn. Pzt) # %B - Ay adı, tam (örn. Ocak) # %b - Ay adı, kısa (örn. Oca) # %p - AM/PM # %Z - Zaman dilimi adı # %c - Yerel tarih ve zaman gösterimi # %x - Yerel tarih gösterimi # %X - Yerel zaman gösterimi

4. Calendar Modülü

Calendar modülü, takvim ile ilgili çeşitli fonksiyonlar sağlar.

import calendar # Bir ayın takvimini metin olarak alma print("2023 Aralık ayı takvimi:") print(calendar.month(2023, 12)) # Bir yılın tüm aylarının takvimini alma # print(calendar.calendar(2023)) # Çok uzun olduğu için yoruma aldık # Bir ayın günlerini liste olarak alma print("2023 Aralık ayının günleri:") print(calendar.monthcalendar(2023, 12)) # Haftanın ilk gününü ayarlama (0: Pazartesi, 6: Pazar) calendar.setfirstweekday(6) # İlk gün Pazar olarak ayarlandı print("Pazar başlangıçlı 2023 Aralık ayı:") print(calendar.month(2023, 12)) # Bir yılın belirli bir ayının ilk günü ilk_gun = calendar.monthrange(2023, 12)[0] print("Aralık 2023'ün ilk gününün haftanın kaçıncı günü olduğu:", ilk_gun) # Bir ayın kaç gün olduğu gun_sayisi = calendar.monthrange(2023, 12)[1] print("Aralık 2023'te kaç gün var:", gun_sayisi) # Bir yılın artık yıl olup olmadığını kontrol etme print("2023 artık yıl mı?", calendar.isleap(2023)) print("2024 artık yıl mı?", calendar.isleap(2024))

5. Zaman Dilimleri

Zaman dilimleriyle çalışmak için pytz kütüphanesini kullanabilirsiniz. Bu kütüphane standart kütüphanenin bir parçası değildir, ayrıca yüklemeniz gerekir.

# Önce pytz kütüphanesini yüklemeniz gerekir: # pip install pytz import datetime import pytz # Mevcut UTC zamanı utc_simdi = datetime.datetime.now(pytz.UTC) print("UTC şu an:", utc_simdi) # Farklı zaman dilimlerindeki zamanları elde etme istanbul = pytz.timezone('Europe/Istanbul') istanbul_simdi = utc_simdi.astimezone(istanbul) print("İstanbul'da şu an:", istanbul_simdi) new_york = pytz.timezone('America/New_York') ny_simdi = utc_simdi.astimezone(new_york) print("New York'ta şu an:", ny_simdi) tokyo = pytz.timezone('Asia/Tokyo') tokyo_simdi = utc_simdi.astimezone(tokyo) print("Tokyo'da şu an:", tokyo_simdi) # Tüm zaman dilimlerini listeleme # for tz in pytz.all_timezones: # print(tz) # Çok uzun olduğu için yoruma aldık

Tarih İşleme Örnekleri

İşte tarih ve zaman işlemleri için daha fazla pratik örnek:

from datetime import datetime, timedelta # Doğum günü hesaplama def dogum_gunu_hesapla(dogum_tarihi): """Bir sonraki doğum gününe kaç gün kaldığını hesaplar""" bugun = datetime.now() dogum = datetime.strptime(dogum_tarihi, "%d/%m/%Y") # Bu yılki doğum günü bu_yil_dogum = datetime(bugun.year, dogum.month, dogum.day) # Eğer bu yılki doğum günü geçtiyse, gelecek yıla bak if bu_yil_dogum < bugun: bu_yil_dogum = datetime(bugun.year + 1, dogum.month, dogum.day) # Kalan gün sayısı kalan_gun = (bu_yil_dogum - bugun).days return kalan_gun # Yaş hesaplama def yas_hesapla(dogum_tarihi): """Yaşı yıl olarak hesaplar""" bugun = datetime.now() dogum = datetime.strptime(dogum_tarihi, "%d/%m/%Y") yas = bugun.year - dogum.year # Eğer bu yılki doğum günü henüz gelmediyse yaşı bir azalt if (bugun.month, bugun.day) < (dogum.month, dogum.day): yas -= 1 return yas # İki tarih arasındaki iş günlerini hesaplama def is_gunleri_hesapla(baslangic, bitis): """İki tarih arasındaki iş günlerini (Pazartesi-Cuma) hesaplar""" baslangic_tarih = datetime.strptime(baslangic, "%d/%m/%Y") bitis_tarih = datetime.strptime(bitis, "%d/%m/%Y") # Hafta içi günleri sayma (0: Pazartesi, 6: Pazar) is_gunu_sayisi = 0 current_date = baslangic_tarih while current_date <= bitis_tarih: if current_date.weekday() < 5: # 0-4 hafta içi günleri temsil eder is_gunu_sayisi += 1 current_date += timedelta(days=1) return is_gunu_sayisi # Örnekleri kullanalım print("Doğum gününe kalan gün:", dogum_gunu_hesapla("15/05/1990")) print("Yaş:", yas_hesapla("15/05/1990")) print("İş günleri:", is_gunleri_hesapla("01/01/2023", "31/01/2023"))

İpucu: Zaman Performansı

Eğer kodunuzda zaman ölçümü yapmanız gerekiyorsa, time.time() yerine daha hassas olan time.perf_counter() fonksiyonunu kullanabilirsiniz.

import time # Bir fonksiyonun çalışma süresini ölçme baslangic = time.perf_counter() # Ölçülecek işlem for i in range(1000000): pass bitis = time.perf_counter() gecen_sure = bitis - baslangic print(f"İşlem {gecen_sure:.6f} saniye sürdü")
