Python'da Liste/Sözlük Üreteçleri

Bu dersimizde Python'da liste üreteçleri (list comprehensions) ve sözlük üreteçleri (dictionary comprehensions) nedir, nasıl kullanılır ve neden önemli olduklarını öğreneceksiniz.

Liste Üreteçleri (List Comprehensions) Nedir?

Liste üreteçleri, Python'da liste oluşturmak için kullanılan zarif ve güçlü bir sözdizimi özelliğidir. For döngüleri ve if koşullarını tek bir satırda birleştirerek liste oluşturmanızı sağlar.

Liste üreteçleri geleneksel for döngülerine göre daha hızlı ve daha kısa kod yazmanızı sağlar. Özellikle bir listeden başka bir liste üretmek için idealdir.

# Geleneksel yöntem ile bir listenin her öğesinin karesini alma sayilar = [1, 2, 3, 4, 5] kareler = [] for sayi in sayilar: kareler.append(sayi ** 2) print(kareler) # [1, 4, 9, 16, 25] # Liste üreteci kullanarak aynı işlemi yapma sayilar = [1, 2, 3, 4, 5] kareler = [sayi ** 2 for sayi in sayilar] print(kareler) # [1, 4, 9, 16, 25]

Biliyor muydunuz?

Liste üreteçleri, geleneksel döngülere göre daha hızlı çalışır çünkü C düzeyinde optimize edilmiştir ve döngü işlemlerini Python yorumlayıcısının içinde yapar.

Liste Üreteci Sözdizimi

Bir liste üretecinin temel sözdizimi şu şekildedir:

[ifade for öğe in sıralı_yapı]

Koşullu ifadeler ekleyerek daha karmaşık liste üreteçleri oluşturabilirsiniz:

[ifade for öğe in sıralı_yapı if koşul]

Birden fazla koşul ve döngü de ekleyebilirsiniz:

[ifade for öğe1 in sıralı_yapı1 if koşul1 for öğe2 in sıralı_yapı2 if koşul2 ...]

Liste Üreteci Örnekleri

# Basit liste üreteci - 1'den 10'a kadar olan sayıların kareleri kareler = [x**2 for x in range(1, 11)] print(kareler) # [1, 4, 9, 16, 25, 36, 49, 64, 81, 100] # Koşullu liste üreteci - 1'den 20'ye kadar olan çift sayıların kareleri cift_kareler = [x**2 for x in range(1, 21) if x % 2 == 0] print(cift_kareler) # [4, 16, 36, 64, 100, 144, 196, 256, 324, 400] # İç içe döngülerle liste üreteci - iki listenin tüm kombinasyonları harfler = ['a', 'b', 'c'] sayilar = [1, 2, 3] kombinasyonlar = [(harf, sayi) for harf in harfler for sayi in sayilar] print(kombinasyonlar) # [('a', 1), ('a', 2), ('a', 3), ('b', 1), ('b', 2), ('b', 3), ('c', 1), ('c', 2), ('c', 3)] # Liste içindeki listeleri düzleştirme matris = [[1, 2, 3], [4, 5, 6], [7, 8, 9]] duz_liste = [eleman for satir in matris for eleman in satir] print(duz_liste) # [1, 2, 3, 4, 5, 6, 7, 8, 9]

Sözlük Üreteçleri (Dictionary Comprehensions) Nedir?

Sözlük üreteçleri, liste üreteçlerine benzer şekilde çalışan ancak sözlük (dictionary) üreten yapılardır. Anahtar-değer çiftlerini tek bir satırda oluşturmanızı sağlar.

# Geleneksel yöntem ile bir sayı dizisinden anahtar-değer çiftleri oluşturma sayilar = [1, 2, 3, 4, 5] kareler_sozluk = {} for sayi in sayilar: kareler_sozluk[sayi] = sayi ** 2 print(kareler_sozluk) # {1: 1, 2: 4, 3: 9, 4: 16, 5: 25} # Sözlük üreteci kullanarak aynı işlemi yapma sayilar = [1, 2, 3, 4, 5] kareler_sozluk = {sayi: sayi ** 2 for sayi in sayilar} print(kareler_sozluk) # {1: 1, 2: 4, 3: 9, 4: 16, 5: 25}

Sözlük Üreteci Sözdizimi

Bir sözlük üretecinin temel sözdizimi şu şekildedir:

{anahtar_ifadesi: değer_ifadesi for öğe in sıralı_yapı}

Koşullu ifadelerle daha karmaşık sözlük üreteçleri oluşturabilirsiniz:

{anahtar_ifadesi: değer_ifadesi for öğe in sıralı_yapı if koşul}

Sözlük Üreteci Örnekleri

# Basit sözlük üreteci - sayıların kendisi ve kareleri kareler_sozluk = {x: x**2 for x in range(1, 6)} print(kareler_sozluk) # {1: 1, 2: 4, 3: 9, 4: 16, 5: 25} # Koşullu sözlük üreteci - sadece çift sayıların kareleri cift_kareler = {x: x**2 for x in range(1, 11) if x % 2 == 0} print(cift_kareler) # {2: 4, 4: 16, 6: 36, 8: 64, 10: 100} # Mevcut bir sözlükten yeni sözlük üretme fiyatlar = {'elma': 5, 'armut': 7, 'muz': 6, 'portakal': 8} zam_yapilmis = {meyve: fiyat * 1.2 for meyve, fiyat in fiyatlar.items()} print(zam_yapilmis) # {'elma': 6.0, 'armut': 8.4, 'muz': 7.2, 'portakal': 9.6} # Koşullu sözlük üreteci - belirli bir fiyatın üstündeki ürünler pahalı_meyveler = {meyve: fiyat for meyve, fiyat in fiyatlar.items() if fiyat > 6} print(pahalı_meyveler) # {'armut': 7, 'portakal': 8}

Küme Üreteçleri (Set Comprehensions)

Python'da liste ve sözlük üreteçlerine ek olarak, küme (set) üreteçleri de bulunmaktadır. Küme üreteçleri, benzersiz elemanlardan oluşan bir küme oluşturur.

# Küme üreteci - 1'den 10'a kadar olan sayıların kareleri kare_kume = {x**2 for x in range(1, 11)} print(kare_kume) # {64, 1, 4, 36, 100, 9, 16, 49, 81, 25} (sıra garanti edilmez) # Koşullu küme üreteci - 20'den küçük çift sayıların kareleri cift_kare_kume = {x**2 for x in range(1, 21) if x % 2 == 0} print(cift_kare_kume) # {256, 4, 64, 36, 100, 16, 144, 196, 324, 400} (sıra garanti edilmez)

İç İçe Üreteçler

Liste, sözlük ve küme üreteçleri içinde birden fazla for döngüsü kullanabilirsiniz. Bu, karmaşık veri yapıları oluşturmak veya çok boyutlu yapıları düzleştirmek için kullanışlıdır.

# İç içe liste üreteci matris = [[1, 2, 3], [4, 5, 6], [7, 8, 9]] kare_matris = [[x**2 for x in satir] for satir in matris] print(kare_matris) # [[1, 4, 9], [16, 25, 36], [49, 64, 81]] # İki listeyi birleştiren sözlük üreteci isimler = ['ali', 'veli', 'ayşe'] yaslar = [25, 30, 22] kisi_bilgileri = {isim: yas for isim, yas in zip(isimler, yaslar)} print(kisi_bilgileri) # {'ali': 25, 'veli': 30, 'ayşe': 22}

Üretecler (Generators) vs. Liste Üreteçleri

Üretecler (generators), liste üreteçlerine benzer fakat tüm listeyi bir kerede belleğe almak yerine, elemanları teker teker üretirler. Bu, özellikle büyük veri setleriyle çalışırken bellek kullanımını optimize eder.

# Liste üreteci - tüm elemanlar belleğe alınır liste_ureteci = [x**2 for x in range(1000000)] # Belleğin çoğunu kullanır # Üretec ifadesi - elemanlar gerektiğinde üretilir (lazy evaluation) uretec = (x**2 for x in range(1000000)) # Çok az bellek kullanır # Üreteç elemanlarını kullanma print(next(uretec)) # 0 print(next(uretec)) # 1 print(next(uretec)) # 4 # Üretecin tüm elemanlarını döngüde kullanma uretec = (x**2 for x in range(5)) for deger in uretec: print(deger)

Ne Zaman Üretec Kullanmalı?

Üretec ifadeleri şu durumlarda tercih edilmelidir:

  • Büyük veri setleriyle çalışırken (bellekten tasarruf için)
  • Tüm verilere bir kerede değil, sırayla erişim gerektiğinde
  • Sonsuz diziler oluşturmanız gerektiğinde
  • Çok fazla veri işlerken veya veri akışları (streams) ile çalışırken
# Büyük dosya satırlarını okuma örneği - üretec kullanımı def dosya_satirlari(dosya_adi): with open(dosya_adi, 'r') as dosya: for satir in dosya: yield satir.strip() # Kullanımı for satir in dosya_satirlari('buyuk_dosya.txt'): print(satir) # Her seferinde bir satır işlenir, tüm dosya belleğe alınmaz

Liste/Sözlük Üreteçlerinin Performans Avantajları

Liste ve sözlük üreteçlerinin geleneksel for döngülerine göre performans avantajları vardır:

  • Hız: Daha hızlı çalışırlar çünkü C düzeyinde optimize edilmişlerdir
  • Okunabilirlik: Daha kısa ve daha anlaşılır kod yazmanızı sağlarlar
  • Fonksiyonel programlama: Daha fonksiyonel bir programlama tarzı sunarlar
  • Tek satırda işlem: Karmaşık işlemleri tek bir satırda gerçekleştirebilirsiniz
import time # Performans karşılaştırması baslangic = time.time() kareler_liste = [] for i in range(1000000): kareler_liste.append(i * i) print(f"For döngüsü süresi: {time.time() - baslangic:.4f} saniye") baslangic = time.time() kareler_liste = [i * i for i in range(1000000)] print(f"Liste üreteci süresi: {time.time() - baslangic:.4f} saniye")
